# Курсы u7-school

Образовательная платформа, направленная на подготовку специалистов через курсы и личное менторство.

## О проекте

Проект представляет собой универсальную платформу для обучения, где курсы базируются на материалах из различных открытых и авторитетных источников. Мы не ограничиваемся одним ресурсом, чтобы предоставить студентам наиболее актуальные и качественные знания.

## Инструменты разработки

Для управления жизненным циклом проекта, планирования и реализации задач используется методология [Conductor](https://github.com/epm-dev-kalki/conductor). Все изменения проходят через строгий процесс спецификации и планирования.

## Установка и запуск

Для установки зависимостей:

```bash
bun install
```

Для запуска проекта:

```bash
bun run index.ts
```

---
Проект создан с использованием [Bun](https://bun.com) — быстрого универсального рантайма для JavaScript.
