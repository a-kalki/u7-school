# How TO - Always Show Scrollbars

* * *

Learn how to always show scrollbars with CSS.

* * *

## How To Force / Always Show Scrollbars

Add `overflow: scroll;` to show both the horizontal and vertical scrollbar:

```javascript
body {  overflow: scroll; /* Show scrollbars */}
```

To only show the vertical scrollbar, or only the horizontal scrollbar, use `overflow-y` or `overflow-x`:

```javascript
body {  overflow-y: scroll; /* Show vertical scrollbar */  overflow-x: scroll; /* Show horizontal scrollbar */}
```

**Tip:** To learn more about the `overflow` property, go to our [CSS Overflow Tutorial](https://www.w3schools.com/css/css_overflow.asp) or [CSS overflow Property Reference](https://www.w3schools.com/cssref/pr_pos_overflow.asp).
