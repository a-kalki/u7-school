## How To Create Rounded Images

##### Step 1) Add HTML:

```javascript
<img src="img_avatar.png" alt="Avatar">
```

* * *

##### Step 2) Add CSS:

Use the `border-radius` property to add rounded corners to an image. 50% will make the image circular:

```javascript
img {  border-radius: 50%;}
```

Go to our [CSS Images Tutorial](https://www.w3schools.com/css/css3_images.asp) to learn more about how to style images.
