# How TO - Notes

* * *

Learn how to create "notes" with CSS.

* * *

## Notes

Notes can be used to notify the user about something special: danger, success, information or warning

**Danger!** Some text...

**Success!** Some text...

**Info!** Some text...

**Warning!** Some text...

[Try it Yourself »](https://www.w3schools.com/howto/tryit.asp?filename=tryhow_css_danger)

* * *

## How To Create Notes

##### Step 1) Add HTML:

```javascript
<div class="danger">  <p><strong>Danger!</strong> Some text...</p></div>
```

* * *

##### Step 2) Add CSS:

```javascript
.danger {  background-color: #ffdddd;  border-left: 6px solid #f44336;}
```

**Tip:** Also check out [Alerts](howto_js_alert.asp.html).
