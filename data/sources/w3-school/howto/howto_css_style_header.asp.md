## Create a Header

##### Step 1) Add HTML:

```javascript
<div class="header">  <h1>Header</h1>  <p>My supercool header</p></div>
```

* * *

* * *

##### Step 2) Add CSS:

Style the header with a large padding, centered text, a specific background-color and a big sized text:

```javascript
.header {  padding: 60px;  text-align: center;  background: #1abc9c;  color: white;  font-size: 30px;}
```
