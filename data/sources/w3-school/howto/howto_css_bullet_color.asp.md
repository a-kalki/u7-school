# How TO - Change Bullet Color of Lists

* * *

Learn how to change bullet colors for lists with CSS.

* * *

## Change Bullet Color

*   Adele
*   Agnes
*   Billy
*   Bob

[Try it Yourself »](https://www.w3schools.com/howto/tryit.asp?filename=tryhow_css_bullet_color)

* * *

##### Step 1) Add HTML:

Create a basic list:

```javascript
<ul>  <li>Adele</li>  <li>Agnes</li>  <li>Billy</li>  <li>Bob</li></ul>
```

##### Step 2) Add CSS:

Use the ::marker selector to change the bullet color of a list item:

```javascript
li::marker {  color: red;}
```
