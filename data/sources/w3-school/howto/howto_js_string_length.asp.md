# How TO - Get The Length of a String

* * *

* * *

## String Length

The `length` property returns the length of a string:

```javascript
var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";var len = txt.length;
```

Read more about strings in our [JavaScript Strings Tutorial](https://www.w3schools.com/js/js_strings.asp).

Read more about the length property in our [JavaScript String length Reference](https://www.w3schools.com/jsref/jsref_length_string.asp).
