# How TO - Color Picker

* * *

Learn how to create a color picker in HTML.

* * *

## Color picker

Pick a color: 

You can use the `input type="color"` attribute to create a color picker:

```javascript
<label for="favcolor">Select your favorite color:</label><input type="color" id="favcolor" value="#ff0000">
```

Note that you have to use hex values inside the value property. The default value is black (#000000) if nothing is used. You can learn more about HEX values in our [HTML Colors Tutorial](https://www.w3schools.com/html/html_colors.asp).
