# MS Access Functions

* * *

MS Access has many built-in functions.

This reference contains the string, numeric, and date functions in MS Access.

* * *

## MS Access String Functions

Function

Description

[Asc](func_msaccess_asc.asp.html)

Returns the ASCII value for the specific character

[Chr](func_msaccess_chr.asp.html)

Returns the character for the specified ASCII number code

[Concat with &](func_msaccess_concat.asp.html)

Adds two or more strings together

[CurDir](func_msaccess_curdir.asp.html)

Returns the full path for a specified drive

[Format](func_msaccess_format.asp.html)

Formats a value with the specified format

[InStr](func_msaccess_instr.asp.html)

Gets the position of the first occurrence of a string in another

[InstrRev](func_msaccess_instrrev.asp.html)

Gets the position of the first occurrence of a string in another, from the end of string

[LCase](func_msaccess_lcase.asp.html)

Converts a string to lower-case

[Left](func_msaccess_left.asp.html)

Extracts a number of characters from a string (starting from left)

[Len](func_msaccess_len.asp.html)

Returns the length of a string

[LTrim](func_msaccess_ltrim.asp.html)

Removes leading spaces from a string

[Mid](func_msaccess_mid.asp.html)

Extracts some characters from a string (starting at any position)

[Replace](func_msaccess_replace.asp.html)

Replaces a substring within a string, with another substring, a specified number of times

[Right](func_msaccess_right.asp.html)

Extracts a number of characters from a string (starting from right)

[RTrim](func_msaccess_rtrim.asp.html)

Removes trailing spaces from a string

[Space](func_msaccess_space.asp.html)

Returns a string of the specified number of space characters

[Split](func_msaccess_split.asp.html)

Splits a string into an array of substrings

[Str](func_msaccess_str.asp.html)

Returns a number as string

[StrComp](func_msaccess_strcomp.asp.html)

Compares two strings

[StrConv](func_msaccess_strconv.asp.html)

Returns a converted string

[StrReverse](func_msaccess_strreverse.asp.html)

Reverses a string and returns the result

[Trim](func_msaccess_trim.asp.html)

Removes both leading and trailing spaces from a string

[UCase](func_msaccess_ucase.asp.html)

Converts a string to upper-case
