## CSS Text Color

The `[color](https://www.w3schools.com/cssref/pr_text_color.php)` property is used to set the color of the text. The color is specified by:

*   a color name - like "red"
*   a HEX value - like "#ff0000"
*   an RGB value - like "rgb(255,0,0)"

Look at [CSS Color Values](https://www.w3schools.com/cssref/css_colors_legal.asp) for a complete list of possible color values.

The default text color for a page is defined in the body selector.

```javascript
body {  color: blue;}h1 {  color: green;}h2 {  color: red;}
```

## Text Color and Background Color

In this example, we define both the `[background-color](https://www.w3schools.com/cssref/pr_background-color.php)` property and the `[color](https://www.w3schools.com/cssref/pr_text_color.php)` property:

```javascript
body {  background-color: lightgrey;  color: blue;}h1 {  background-color: black;  color: white;}div {  background-color: blue;  color: white;}
```

**Important:** High contrast is very important for people with vision problems. So, always ensure that the contrast between the text color and the background color (or background image) is good!
