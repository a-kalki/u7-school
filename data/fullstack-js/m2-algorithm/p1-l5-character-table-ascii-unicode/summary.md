# Таблица символов — краткая выжимка

⏱ **Видео:** ~25 мин

- **Код символа** — число, которым компьютер кодирует символ: `'A'` = 65, `'a'` = 97, `'0'` = 48, `' '` = 32
- **ASCII** — 128 символов (0–127); **Unicode** — 149 000+ символов (ASCII — подмножество)
- **Строчные больше заглавных:** `'a'` (97) > `'A'` (65) → `'Zoo' < 'apple'`
- **Пробел (32) меньше всего:** `' a' < 'aa'`, `'aa' < 'aa '`
- **Лексикографический порядок** — посимвольное сравнение слева направо по кодам:
  - `'ab' < 'ac'` (первые равны, `'b'` < `'c'`)
  - `'aa' < 'aaa'` (символы кончились в первой → она меньше)
  - `'b' > 'aaaaa'` (на первом символе `'b'` > `'a'`)
- **Разрешены** `.charCodeAt()` и `String.fromCodePoint()` — исключение из правил для работы с кодами символов
- В функциях сравнения используем **прямое сравнение** одиночных символов через `<` и `>`
