# Функция replace(str, search, replacement) — замена первого вхождения подстроки

**Краткое содержание:**
Урок посвящён реализации функции `replace(str, search, replacement)` — замена первого вхождения `search` на `replacement`. Студент использует уже реализованную `indexOf` для поиска и собирает результат из трёх частей.

### Алгоритм

Функция `replace(str, search, replacement)` должна найти первое вхождение подстроки `search` в `str` и заменить его на `replacement`. Если `search` не найден — вернуть исходную строку без изменений.

**Шаги алгоритма:**
1. Проверить типы: все три аргумента должны быть string. Если нет — `throw new TypeError`
2. Если `search` — пустая строка (`''`) — поведение зависит от реализации. В стандартном JS `''` заменяется перед первым символом. Примем решение: если `search === ''`, вернуть `replacement + str`
3. Найти индекс первого вхождения через `indexOf(str, search)` (уже реализована)
4. Если `index === -1` — вернуть `str` без изменений
5. Собрать результат из трёх частей:
   - Левая часть: от 0 до `index`
   - Замена: `replacement`
   - Правая часть: от `index + len(search)` до конца
6. Вернуть склеенную строку

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Замена в середине | `replace('hello world', 'world', 'everyone')` | `'hello everyone'` |
| Замена в начале | `replace('hello world', 'hello', 'hi')` | `'hi world'` |
| Замена в конце | `replace('hello world', 'world', 'earth')` | `'hello earth'` |
| Подстрока не найдена | `replace('hello', 'xyz', 'abc')` | `'hello'` |
| Замена на более длинную | `replace('hi', 'i', 'ello')` | `'hello'` |
| Замена на более короткую | `replace('hello', 'ello', 'i')` | `'hi'` |
| Замена на пустую строку | `replace('hello', 'll', '')` | `'heo'` |
| Пустая поисковая строка | `replace('hello', '', 'X')` | `'Xhello'` |
| Пустая исходная строка | `replace('', 'abc', 'X')` | `''` |
| Множественные вхождения — только первое | `replace('banana', 'na', 'NA')` | `'baNAna'` |
| Кириллица | `replace('привет мир', 'мир', 'всем')` | `'привет всем'` |
| Первый аргумент не строка | `replace(123, 'a', 'b')` | `TypeError` |
| Второй аргумент не строка | `replace('hello', 123, 'b')` | `TypeError` |
