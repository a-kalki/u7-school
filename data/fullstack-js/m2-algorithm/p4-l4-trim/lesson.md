# Функция trim(str) — удаление пробелов с начала и конца строки

**Краткое содержание:**
Урок посвящён реализации функции `trim(str)` — удаление пробельных символов с начала и конца строки. Студент самостоятельно пишет тесты и реализацию.

### Алгоритм

Функция `trim(str)` должна вернуть новую строку, из которой удалены все пробелы в начале и конце. Пробелы внутри строки остаются нетронутыми.

**Шаги алгоритма:**
1. Проверить тип: `str` должен быть string. Если нет — `throw new TypeError`
2. Найти `begin` — индекс первого символа, который НЕ пробел. Пройти циклом от 0 вперёд, пока `str[i] === ' '`
3. Если `begin === len(str)` — вся строка из пробелов, вернуть `''`
4. Найти `end` — индекс последнего символа, который НЕ пробел. Пройти циклом от `len(str) - 1` назад, пока `str[i] === ' '`
5. Вернуть подстроку от `begin` до `end + 1` (не включая `end + 1`), используя цикл

### Реализация

Студент реализует функцию в файле `trim.js` и тесты в `trim.test.js`, следуя TDD.

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Пробелы с обеих сторон | `trim('  hello  ')` | `'hello'` |
| Пробелы только слева | `trim('  hello')` | `'hello'` |
| Пробелы только справа | `trim('hello  ')` | `'hello'` |
| Без пробелов | `trim('hello')` | `'hello'` |
| Только пробелы | `trim('   ')` | `''` |
| Пустая строка | `trim('')` | `''` |
| Пробелы внутри строки | `trim('  he llo  ')` | `'he llo'` |
| Табуляция и перевод строки (только пробел) | `trim(' \t hello \n ')` | `'\t hello \n'` |
| Кириллица с пробелами | `trim('  привет  ')` | `'привет'` |
| Аргумент не строка | `trim(123)` | `TypeError` |
| Аргумент null | `trim(null)` | `TypeError` |
