# Шаги урока: Функция trim()

---

### Реализация trim()

**kind:** `text`

Реализуй функцию `trim(str)` — удаление пробелов с начала и конца строки. Создай файлы `trim.js` и `trim.test.js`, следуй TDD.

**Функция принимает:**
- `str`: `string` — исходная строка

**Функция возвращает:** `string` — строка без пробелов в начале и конце. Пробелы внутри строки остаются.

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна удалить пробелы с обеих сторон: `('  hello  ')` → `'hello'`
- Должна удалить пробелы только слева: `('  hello')` → `'hello'`
- Должна удалить пробелы только справа: `('hello  ')` → `'hello'`
- Должна вернуть строку без изменений если пробелов нет: `('hello')` → `'hello'`
- Должна вернуть `''` для строки из одних пробелов: `('   ')` → `''`
- Должна вернуть `''` для пустой строки: `('')` → `''`
- Должна сохранить пробелы внутри строки: `('  he llo  ')` → `'he llo'`
- Должна удалить только пробелы, не `\t` и `\n`: `(' \t hello \n ')` → `'\t hello \n'`
- Должна работать с кириллицей: `('  привет  ')` → `'привет'`
- Должна выбросить `TypeError` если аргумент не строка: `(123)` → ошибка
- Должна выбросить `TypeError` если аргумент `null`
