# trim(str) — краткая выжимка

- **Удаляет пробелы** с начала и конца строки, не трогая середину
- **Алгоритм:** два прохода — найти первый не-пробел слева, найти первый не-пробел справа
- Если вся строка из пробелов → `''`
- Удаляются только пробелы (`' '`, код 32), не табуляции и переводы строк
- `throw new TypeError` если аргумент не string
