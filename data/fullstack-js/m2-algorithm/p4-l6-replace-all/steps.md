# Шаги урока: Функция replaceAll()

---

### Реализация replaceAll()

**kind:** `text`

Реализуй функцию `replaceAll(str, search, replacement)` — замена **всех** вхождений подстроки.

**Функция принимает:**
- `str`: `string` — исходная строка
- `search`: `string` — что ищем
- `replacement`: `string` — на что заменяем

**Функция возвращает:** `string` — строка, в которой все вхождения `search` заменены на `replacement`. Если `search` не найден — исходная строка.

**Ключевая тонкость — без рекурсивной замены:**
Заменённый текст НЕ участвует в дальнейшем поиске. Например:
- `'aaa'.replaceAll('aa', 'ba')` → `'baa'` (не `'ba'`)

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна заменить одно вхождение: `('hello', 'll', 'LL')` → `'heLLo'`
- Должна заменить несколько вхождений: `('banana', 'na', 'NA')` → `'baNANA'`
- Должна не учитывать подставленную строку: `('aaa', 'aa', 'ba')` → `'baa'`
- Должна вернуть исходную строку если search не найден: `('hello', 'eli', 'hhh')` → `'hello'`
- Должна работать с заменой символе на строку: `('a', 'a', 'hello')` → `'hello'`
- Должна работать с заменой на более длинную: `('hi', 'i', 'ello')` → `'hello'`
- Должна работать с заменой на пустую строку: `('hello', 'l', '')` → `'heo'`
- Должна вернуть `''` для пустой исходной строки: `('', 'abc', 'X')` → `''`
- Должна вставить replacement вокруг каждого символа при пустом search: `('abc', '', 'X')` → `'XaXbXcX'`
- Должна выбросить `TypeError` если любой аргумент не строка
