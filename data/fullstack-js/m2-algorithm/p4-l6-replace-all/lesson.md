# Функция replaceAll(str, search, replacement) — замена всех вхождений подстроки

**Краткое содержание:**
Урок посвящён реализации функции `replaceAll(str, search, replacement)` — замена всех вхождений `search` на `replacement`. Ключевая тонкость: замены не должны применяться рекурсивно к уже заменённому тексту.

### Алгоритм

Функция `replaceAll(str, search, replacement)` должна найти **все** вхождения подстроки `search` в `str` и заменить каждое на `replacement`. Важно: заменённый текст не участвует в дальнейшем поиске.

**Шаги алгоритма:**
1. Проверить типы: все три аргумента должны быть string. Если нет — `throw new TypeError`
2. Если `search` — пустая строка — поведение особое. В стандартном JS это вставляет `replacement` между каждым символом. Примем решение: если `search === ''`, вернуть результат посимвольной вставки
3. Пройти циклом по `str`, собирая результат:
   - На каждой позиции `i` проверить: совпадает ли `search` с подстрокой начиная с `i`
   - Если совпадает — добавить `replacement` к результату и сдвинуть `i` на `len(search)` (чтобы не обрабатывать заменённое)
   - Если не совпадает — добавить `str[i]` к результату и сдвинуть `i` на 1
4. Вернуть собранную строку

> **Альтернативный подход:** использовать `indexOf` в цикле — найти первое вхождение, вырезать левую часть + replacement, продолжить поиск в оставшейся правой части. Оба подхода принимаются.

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Одно вхождение | `replaceAll('hello', 'll', 'LL')` | `'heLLo'` |
| Несколько вхождений | `replaceAll('banana', 'na', 'NA')` | `'baNANA'` |
| Подряд идущие вхождения | `replaceAll('aaaa', 'aa', 'b')` | `'bb'` |
| Нет вхождений | `replaceAll('hello', 'xyz', 'abc')` | `'hello'` |
| Замена на более длинную | `replaceAll('a', 'a', 'hello')` | `'hello'` |
| Замена на пустую | `replaceAll('hello', 'l', '')` | `'heo'` |
| Пустая исходная строка | `replaceAll('', 'abc', 'X')` | `''` |
| Пустой search | `replaceAll('abc', '', 'X')` | `'XaXbXcX'` |
| Пересекающиеся вхождения (не рекурсивно) | `replaceAll('aaa', 'aa', 'b')` | `'ba'` |
| Пробелы | `replaceAll('a b a b', ' a', '')` | `'b b'` |
| Аргумент не строка | `replaceAll(123, 'a', 'b')` | `TypeError` |
