# Пуш ветки и создание PR — краткая выжимка

- **Перед пушем:** `bun test` зелёные, `git status` clean, ты в `feat/substring-search`
- **Пуш:** `git push -u origin feat/substring-search` — флаг `-u` запоминает связь с удалённой веткой
- **PR:** GitHub → Compare & pull request → base: `main`, compare: `feat/substring-search`
- **Заголовок:** кратко и по делу
- **Описание:** что сделано + как запустить тесты
- **Reviewers:** выбрать 2 ревьюеров
- **Pin:** закрепить сообщение со ссылкой на PR в чате
