### Реализация slice()

**kind:** `text`

> Убедись, что ты в ветке `feat/string-transform`

Реализуй функцию `slice(str, start, end)` — извлечение подстроки с поддержкой отрицательных индексов. Создай файлы `slice.js` и `slice.test.js`, следуй TDD.

**Функция принимает:**
- `str`: `string` — исходная строка
- `start`: `number` — начальный индекс (включая). Если отрицательный — отсчёт с конца
- `end`: `number` (необязательный) — конечный индекс (не включая). Если не передан — до конца строки. Если отрицательный — отсчёт с конца

**Функция возвращает:** `string` — извлечённая подстрока.

**Отличия от substring:**
- Отрицательные индексы работают (`-1` = последний символ)
- `start` и `end` **не** меняются местами
- `end` можно не передавать (по умолчанию — `len(str)`)

**Напоминание:**
- Создай `slice.js` с заглушкой, добавь JSDoc
- Создай `slice.test.js`, следуй TDD: RED → GREEN → REFACTOR

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна вернуть `'hello'` для `('hello', 0, 5)`
- Должна вернуть `'ell'` для `('hello', 1, 4)`
- Должна вернуть `'ell'` для `('hello', 2, 4)`
- Должна вернуть `'llo'` для `('hello', 2)` — без end, до конца
- Должна вернуть `'h'` для `('hello', 0, 1)`
- Должна вернуть `'l'` для `('hello', 2, 3)`
- Должна вернуть `'o'` для `('hello', 4)`
- Должна вернуть `'llo'` для отрицательного start: `('hello', -3)`
- Должна вернуть `'hell'` для отрицательного end: `('hello', 0, -1)`
- Должна вернуть `'ll'` для обоих отрицательных: `('hello', -3, -1)`
- Должна вернуть `''` если start >= end после нормализации: `('hello', 4, 1)`
- Должна вернуть `''` если start за границей: `('hello', 10, 15)`
- Должна нормализовать выходящий за границы отрицательный start: `('hello', -10, 3)` → `'hel'`
- Должна вернуть `''` для пустой строки: `('', 0, 1)` → `''`
- Должна работать с кириллицей: `('привет', 1, 4)` → `'рив'`
- Должна выбросить `TypeError` если str не строка
- Должна выбросить `TypeError` если 2 арг. не число
- Должна выбросить `TypeError` если 3 арг. указан и не число
