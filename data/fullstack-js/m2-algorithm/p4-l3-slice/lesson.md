# Функция slice(str, start, end) — извлечение подстроки с поддержкой отрицательных индексов

**Краткое содержание:**
Урок посвящён реализации функции `slice(str, start, end)` — извлечение подстроки с поддержкой отрицательных индексов (отсчёт с конца). Студент самостоятельно пишет тесты и реализацию.

### Алгоритм

Функция `slice(str, start, end)` должна вернуть подстроку от `start` до `end` (не включая `end`), с поддержкой отрицательных индексов. В отличие от `substring`, отрицательные индексы работают (отсчёт с конца), а `start` и `end` **не** меняются местами.

**Шаги алгоритма:**
1. Проверить типы: `str` — string, `start` и `end` — number (если переданы). Если нет — `throw new TypeError`
2. Если `end` не передан — установить `end = len(str)`
3. Нормализовать `start`: если отрицательный → `len(str) + start`. Если всё ещё < 0 → 0. Если > `len(str)` → `len(str)`
4. Нормализовать `end`: если отрицательный → `len(str) + end`. Если > `len(str)` → `len(str)`. Если < 0 → 0
5. Если `start >= end` после нормализации — вернуть `''`
6. Пройти циклом от `start` до `end`, собрать символы → вернуть результат

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Обычное извлечение | `slice('hello', 0, 5)` | `'hello'` |
| Из середины | `slice('hello', 1, 4)` | `'ell'` |
| Без end — до конца | `slice('hello', 2)` | `'llo'` |
| Отрицательный start | `slice('hello', -3)` | `'llo'` |
| Отрицательный end | `slice('hello', 0, -1)` | `'hell'` |
| Оба отрицательные | `slice('hello', -3, -1)` | `'ll'` |
| start > end — пустая строка | `slice('hello', 4, 1)` | `''` |
| start за границей | `slice('hello', 10, 15)` | `''` |
| Отрицательный start, выходящий за границы | `slice('hello', -10, 3)` | `'hel'` |
| Пустая строка | `slice('', 0, 1)` | `''` |
| Кириллица | `slice('привет', 1, 4)` | `'рив'` |
| Первый аргумент не строка | `slice(123, 0, 1)` | `TypeError` |
