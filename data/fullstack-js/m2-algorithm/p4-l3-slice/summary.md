# slice(str, start, end) — краткая выжимка

- **Извлечение подстроки** с поддержкой отрицательных индексов (отсчёт с конца)
- **Отрицательный индекс:** `len(str) + index`. `-3` в `'hello'` → `5 + (-3)` = `2` (символ `'l'`)
- **start и end НЕ меняются местами** (в отличие от substring)
- **start >= end** после нормализации → `''`
- `end` не обязателен — по умолчанию до конца строки
