# Функция reverse(str) — переворот строки

**Краткое содержание:**
Урок посвящён реализации функции переворота строки. Студент самостоятельно пишет тесты и реализацию по текстовому описанию задачи и тестовых случаев.

### Алгоритм

Функция `reverse(str)` должна вернуть новую строку, в которой символы исходной строки идут в обратном порядке.

**Шаги алгоритма:**
1. Проверить тип аргумента: должен быть string. Если нет — `throw new TypeError`
2. Создать пустую строку-результат `result = ''`
3. Пройти циклом по символам `str` с конца: от `len(str) - 1` до `0`
4. На каждом шаге добавлять `str[i]` к `result`
5. Вернуть `result`

**Альтернативный подход (без конкатенации):**
Если конкатенация строк (`result += str[i]`) вызывает вопросы (ведь это создаёт новую строку на каждом шаге), можно:
- Создать массив-результат нужной длины
- Заполнить его символами в обратном порядке
- Вернуть как строку (но функция `join` ещё не реализована, так что конкатенация — рабочий вариант)

### Реализация

Студент реализует функцию в файле `reverse.js` и тесты в `reverse.test.js`, следуя TDD-подходу (red → green → refactor).

**Напоминание:**
- Создай файл `reverse.js` с заглушкой: `export function reverse(str) { return ''; }`

- Запусти `bun test` — тест должен упасть (RED)
- Реализуй функцию — тест должен пройти (GREEN)

### Тестовые случаи

Студент должен написать тесты для следующих случаев:

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Обычная строка | `reverse('hello')` | `'olleh'` |
| Пустая строка | `reverse('')` | `''` |
| Один символ | `reverse('a')` | `'a'` |
| Палиндром | `reverse('racecar')` | `'racecar'` |
| Строка с пробелами | `reverse('a b c')` | `'c b a'` |
| Кириллица | `reverse('привет')` | `'тевирп'` |
| Аргумент не строка | `reverse(123)` | `TypeError` |
| Аргумент null | `reverse(null)` | `TypeError` |
