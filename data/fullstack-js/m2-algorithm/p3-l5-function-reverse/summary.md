# reverse(str) — краткая выжимка

- **Алгоритм:** цикл с конца строки (`len(str) - 1` до `0`), добавляем каждый символ к результату
- Пустая строка → `''`
- Один символ → тот же символ
- Палиндром → не меняется
- Работает с пробелами и кириллицей
- `throw new TypeError` если аргумент не string
