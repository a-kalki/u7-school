---

### Реализация reverse()

**kind:** `text`

> Убедись, что ты в ветке `feat/substring-search`: `git switch feat/substring-search`

Реализуй функцию `reverse(str)` — переворот строки. Создай файлы `reverse.js` и `reverse.test.js`, следуй TDD.

**Функция принимает:**
- `str`: `string` — исходная строка

**Функция возвращает:** `string` — новая строка с символами в обратном порядке.

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна перевернуть строку: `reverse('hello')` → `'olleh'`
- Должна вернуть `''` для пустой строки
- Должна вернуть `'a'` для строки из одного символа
- Должна вернуть ту же строку для палиндрома: `reverse('racecar')` → `'racecar'`
- Должна перевернуть строку с пробелами: `reverse('a b c')` → `'c b a'`
- Должна работать с кириллицей: `reverse('привет')` → `'тевирп'`
- Должна выбросить `TypeError` если аргумент не строка (`123`)
- Должна выбросить `TypeError` если аргумент `null`
