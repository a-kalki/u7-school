# Pull Request — краткая выжимка

⏱ **Видео:** ~15 мин

- **Pull Request (PR)** — предложение изменений с код-ревью перед попаданием в основную ветку
- **GitHub Flow:** создать ветку → коммиты → открыть PR → обсудить → слить → удалить ветку
- **Создание PR:** GitHub → Pull requests → New pull request → выбрать ветки → заголовок + описание
- **Хорошее описание:** что сделано, как запустить, на что обратить внимание
- **Files changed** — diff: красные (удалено) и зелёные (добавлено) строки
- PR создан один раз; новые коммиты в ту же ветку обновляют PR автоматически
