# Обработка ошибок — краткая выжимка

⏱ **Видео:** ~18 мин

- `throw new TypeError(...)` — выбрасывает ошибку и останавливает выполнение функции
- **Типы ошибок:** `TypeError` (неверный тип), `RangeError` (выход за границы), `Error` (универсальный)
- `typeof arg !== 'string'` — проверка типа перед выполнением логики
- `try { ... } catch (error) { ... }` — перехват ошибок; программа не падает
- В тестах try-catch используют для проверки, что функция выбрасывает ожидаемую ошибку
- `error.message` — текст ошибки; `error instanceof TypeError` — проверка типа ошибки
