# Правила ревью — краткая выжимка

⏱ **Видео:** ~15 мин

- **Код-ревью** — обязательная проверка кода коллегами перед мержем. Стандарт индустрии
- **Добавление ревьюеров:** PR → Reviewers (правая панель) → ввести username
- **Правило модуля:** нужно 2 аппрува для мержа
- **Предложение:** «Предлагаю / Можно сделать X, потому что Y» — автор сам решает
- **Критическое:** «Нужно / Необходимо сделать X, потому что Y» — обязательно к исправлению
- Только предложения → **Approve** ✅ | Есть критические → **Request changes** ❌
- **Что проверяем:** корректность, граничные случаи, правила модуля, читаемость, тесты
