# Функция upperCase(str) — перевод в верхний регистр через коды символов

**Краткое содержание:**
Урок посвящён реализации функции `upperCase(str)` — перевод строчных букв в заглавные с использованием кодов символов. Студент применяет знания таблицы символов: функция работает и для латиницы (`'a'`–`'z'` → `'A'`–`'Z'`), и для кириллицы (`'а'`–`'я'` → `'А'`–`'Я'`). В обоих случаях разница между строчной и заглавной — 32.

### Алгоритм

Функция `upperCase(str)` должна вернуть новую строку, в которой все строчные буквы (латиница и кириллица) заменены на заглавные. Остальные символы (уже заглавные буквы, цифры, пробелы, знаки) остаются без изменений.

**Шаги алгоритма:**
1. Проверить тип: `str` должен быть string. Если нет — `throw new TypeError`
2. Создать пустую строку-результат
3. Пройти циклом по каждому символу `str[i]`:
   - Получить код символа через `.charCodeAt(0)` (разрешён)
   - Если код в диапазоне 97–122 (строчная латиница `'a'`–`'z'`) **или** 1072–1103 (строчная кириллица `'а'`–`'я'`):
     - Вычесть 32 (97 → 65 = `'A'`, 1072 → 1040 = `'А'`)
     - Преобразовать код обратно в символ через `String.fromCodePoint(код)` (разрешён)
     - Добавить символ к результату
   - Иначе — добавить исходный символ к результату
4. Вернуть результат

**Почему именно вычитание 32:**
- `'a'` = 97, `'A'` = 65, разница = 32
- `'а'` = 1072, `'А'` = 1040, разница = 32
- Для любой строчной буквы (латиница или кириллица): `заглавная = строчная − 32`

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Все строчные латинские | `upperCase('hello')` | `'HELLO'` |
| Уже заглавные | `upperCase('HELLO')` | `'HELLO'` |
| Смешанный регистр | `upperCase('Hello World')` | `'HELLO WORLD'` |
| Строка с цифрами | `upperCase('hello123')` | `'HELLO123'` |
| Строка со знаками | `upperCase('hello!@#')` | `'HELLO!@#'` |
| Пустая строка | `upperCase('')` | `''` |
| Только не-буквы | `upperCase('123!@#')` | `'123!@#'` |
| Кириллица | `upperCase('привет')` | `'ПРИВЕТ'` |
| Смесь латиницы и кириллицы | `upperCase('hello привет')` | `'HELLO ПРИВЕТ'` |
| Уже заглавная кириллица | `upperCase('ПРИВЕТ')` | `'ПРИВЕТ'` |
| Аргумент не строка | `upperCase(123)` | `TypeError` |
