# Шаги урока: Функция upperCase()

---

### Реализация upperCase()

**kind:** `text`

Реализуй функцию `upperCase(str)` — перевод строчных букв в заглавные через коды символов. Работает как для латиницы, так и для кириллицы.

**Функция принимает:**
- `str`: `string` — исходная строка

**Функция возвращает:** `string` — строка, в которой все строчные буквы (`'a'`–`'z'`, `'а'`–`'я'`) заменены на соответствующие заглавные. Остальные символы не меняются.

**Напоминание:**
- Разрешённые методы: `.charCodeAt(0)` и `String.fromCodePoint()`

**Коды символов:**
- `'a'` (97), `'A'` (65);
- `'а'` (1072), `'А'` (1040).
- коды символов `'z', 'Z', 'я', 'Я'`, определите сами через таблицу символов или метод `''.charCodeAt()`.

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна перевести все строчные латинские: `('hello')` → `'HELLO'`
- Должна оставить заглавные без изменений: `('HELLO')` → `'HELLO'`
- Должна работать со смешанным регистром: `('Hello World')` → `'HELLO WORLD'`
- Должна не трогать пробелы: `(' hello 123')` → `' HELLO 123'`
- Должна не менять цифры: `('hello123')` → `'HELLO123'`
- Должна не менять знаки: `('hello!@#')` → `'HELLO!@#'`
- Должна вернуть `''` для пустой строки
- Должна вернуть строку без изменений если нет строчных букв: `('123!@#')` → `'123!@#'`
- Должна перевести кириллицу: `('привет')` → `'ПРИВЕТ'`
- Должна работать со смесью латиницы и кириллицы: `('hello привет')` → `'HELLO ПРИВЕТ'`
- Должна не менять уже заглавные буквы (включая кириллицу): `('ПРИВЕТ')` → `'ПРИВЕТ'`
- Должна выбросить `TypeError` если аргумент не строка
