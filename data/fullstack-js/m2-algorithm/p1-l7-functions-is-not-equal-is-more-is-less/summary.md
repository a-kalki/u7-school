# isNotEqual, isMore, isLess — краткая выжимка

⏱ **Видео:** ~18 мин

- **isNotEqual(a, b):** вернуть `!isEqual(a, b)` — использует уже готовую функцию
- **isMore(a, b):** посимвольное сравнение слева направо
  - Нашли символ где `a[i] > b[i]` → `true`
  - Нашли символ где `a[i] < b[i]` → `false`
  - Символы в `a` кончились, в `b` ещё есть → `false` (a короче)
  - Символы в `b` кончились, в `a` ещё есть → `true` (a длиннее)
  - Все символы равны → `false`
- **isLess(a, b):** `!isMore(a, b) && !isEqual(a, b)` — не больше И не равно
