# Шаги урока: Функции isNotEqual, isMore, isLess

---

### Шаг 1 — Функция isNotEqual(a, b)

**kind:** `text`

Реализуй `isNotEqual(a, b)` — вернуть `true`, если строки НЕ равны.

Тестовые случаи:
- Должна вернуть `true` для разных строк (`'hello'`, `'world'`)
- Должна вернуть `false` для одинаковых строк (`'abc'`, `'abc'`)
- Должна вернуть `true` для строк разной длины (`'hi'`, `'hello'`)
- Должна вернуть `false` для пустых строк (`''`, `''`)
- Должна выбросить `TypeError` если аргумент(ы) не строка

Закоммить:

```
git add is-not-equal.js is-not-equal.test.js
git commit -m "Реализовать isNotEqual"
```

### Шаг 2 — Функция isMore(a, b)

**kind:** `text`

Реализуй `isMore(a, b)` — вернуть `true`, если строка `a` лексикографически **больше** строки `b`.

Тестовые случаи:
- Должна вернуть `true` если `a` явно больше — различие на третьем символе (`'cat'`, `'car'`)
- Должна вернуть `false` если `a` явно меньше (`'car'`, `'cat'`)
- Должна вернуть `false` для равных строк (`'hello'`, `'hello'`)
- Должна вернуть `true` если `a` длиннее, но символы совпадают (`'hello!'`, `'hello'`)
- Должна вернуть `false` если `a` короче, но символы совпадают (`'hello'`, `'hello!'`)
- Должна вернуть `true` если первый символ `a` больше — длина не важна (`'b'`, `'aaaaa'`)
- Должна вернуть `false` для заглавной vs строчной — `'A'`(65) < `'a'`(97) (`'Admin'`, `'admini'`)
- Должна вернуть `false` если `a` начинается с пробела, а `b` с буквы (`' a'`, `'aa'`)
- Должна вернуть `true` если `a` заканчивается пробелом (`'aa '`, `'aa'`)
- Должна вернуть `false` для пустых строк (`''`, `''`)
- Должна вернуть `false` если `a` пустая, `b` нет (`''`, `'a'`)
- Должна выбросить `TypeError` если аргумент не строка

Закоммить:

```
git add is-more.js is-more.test.js
git commit -m "Реализовать isMore — лексикографическое сравнение"
```

### Шаг 3 — Функция isLess(a, b)

**kind:** `text`

Реализуй `isLess(a, b)` — вернуть `true`, если `a` лексикографически **меньше** `b`.

Тестовые случаи:
- Должна вернуть `true` если `a` явно меньше (`'car'`, `'cat'`)
- Должна вернуть `false` если `a` больше (`'cat'`, `'car'`)
- Должна вернуть `false` для равных строк (`'hello'`, `'hello'`)
- Должна вернуть `true` если `a` короче (`'hello'`, `'hello!'`)
- Должна вернуть `true` для заглавной vs строчной (`'A'`, `'a'`)
- Должна вернуть `true` для пустой vs непустой (`''`, `'a'`)
- Должна выбросить `TypeError` если аргумент не строка

Закоммить:

```
git add is-less.js is-less.test.js
git commit -m "Реализовать isLess(a, b)"
```
