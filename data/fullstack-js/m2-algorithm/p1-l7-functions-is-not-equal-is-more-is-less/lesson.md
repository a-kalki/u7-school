# Функции isNotEqual, isMore, isLess

**Краткое содержание:**
Урок посвящён реализации трёх функций сравнения строк. `isNotEqual` — отрицание `isEqual`. `isMore` — лексикографическое сравнение (строго больше). `isLess` — обратное к `isMore`. Студент пишет тесты и реализацию самостоятельно.

---

### 1. Функция isNotEqual(a, b)

**Задача:** вернуть `true`, если строки НЕ равны, и `false` если равны.

**Алгоритм:** использовать уже готовую `isEqual` и вернуть отрицание.

```javascript
function isNotEqual(a, b) {
  return !isEqual(a, b);
}
```

#### Тестовые случаи для isNotEqual

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Разные строки | `isNotEqual('hello', 'world')` | `true` |
| Одинаковые строки | `isNotEqual('abc', 'abc')` | `false` |
| Разная длина | `isNotEqual('hi', 'hello')` | `true` |
| Пустые строки (равны) | `isNotEqual('', '')` | `false` |
| Не строка | `isNotEqual(123, 'hello')` | `TypeError` |

---

### 2. Функция isMore(a, b)

**Задача:** вернуть `true`, если строка `a` лексикографически БОЛЬШЕ строки `b`.

**Алгоритм (посимвольное сравнение):**

1. Проверить типы: оба аргумента должны быть `string`
2. Пройти циклом от 0 до `len(a)` (или до первого различия):
   - Если `i` вышел за пределы `b` (т.е. `b[i] === undefined`) → `a` длиннее и все предыдущие символы равны → `a > b`, вернуть `true`
   - Сравнить `a[i]` и `b[i]`:
     - Если `a[i] > b[i]` → вернуть `true`
     - Если `a[i] < b[i]` → вернуть `false`
     - Если равны → продолжить
3. Если цикл завершился (все символы `a` совпали с началом `b`) → теперь проверить длины:
   - Если `len(a) === len(b)` → строки равны, вернуть `false`
   - Если `len(a) < len(b)` → `a` короче, вернуть `false`

**Примеры по шагам:**

```
isMore('cat', 'car'):
  i=0: 'c' === 'c' → дальше
  i=1: 'a' === 'a' → дальше
  i=2: 't' > 'r' → true ✓

isMore('aa', 'aaa'):
  i=0: 'a' === 'a' → дальше
  i=1: 'a' === 'a' → дальше
  i=2: b[2] === 'a', a[2] === undefined → a кончилась → a < b → false ✓

isMore('b', 'aaaaa'):
  i=0: 'b' > 'a' → true ✓ (несмотря на длину!)

isMore('A', 'a'):
  i=0: 'A' (65) > 'a' (97)? → НЕТ! 65 < 97 → false ✓
```

#### Тестовые случаи для isMore

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Явно больше (по второму символу) | `isMore('cat', 'car')` | `true` |
| Явно меньше | `isMore('car', 'cat')` | `false` |
| Равные строки | `isMore('hello', 'hello')` | `false` |
| Первая длиннее (символы совпадают) | `isMore('hello!', 'hello')` | `true` |
| Вторая длиннее (символы совпадают) | `isMore('hello', 'hello!')` | `false` |
| Первая «явно больше» по первому символу | `isMore('b', 'aaaaa')` | `true` |
| Заглавная vs строчная | `isMore('A', 'a')` | `false` (65 < 97) |
| Пробел в начале | `isMore(' a', 'aa')` | `false` (32 < 97) |
| Пробел в конце | `isMore('aa ', 'aa')` | `true` |
| Пустые строки (равны) | `isMore('', '')` | `false` |
| Пустая vs непустая | `isMore('', 'a')` | `false` |
| Не строка | `isMore(123, 'hello')` | `TypeError` |

### 3. Функция isLess(a, b)

**Задача:** вернуть `true`, если строка `a` лексикографически МЕНЬШЕ строки `b`.

Можно реализовать двумя способами:
- **Способ 1:** написать отдельный алгоритм (аналогично `isMore`, но с обратной логикой)
- **Способ 2 (рекомендуется):** использовать `isMore` и `isEqual` — строка `a` меньше `b`, если `a` не больше `b` И `a` не равна `b`:

```javascript
function isLess(a, b) {
  return !isMore(a, b) && !isEqual(a, b);
}
```

#### Тестовые случаи для isLess

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Явно меньше | `isLess('car', 'cat')` | `true` |
| Явно больше | `isLess('cat', 'car')` | `false` |
| Равные строки | `isLess('hello', 'hello')` | `false` |
| Первая короче (символы совпадают) | `isLess('hello', 'hello!')` | `true` |
| Заглавная vs строчная | `isLess('A', 'a')` | `true` (65 < 97) |
| Пустая vs непустая | `isLess('', 'a')` | `true` |
| Не строка | `isLess(123, 'hello')` | `TypeError` |
