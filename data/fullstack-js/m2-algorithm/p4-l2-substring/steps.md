### Реализация substring()

**kind:** `text`

> Создай новую ветку: `git switch -c feat/string-transform`

Реализуй функцию `substring(str, start, end)` — извлечение подстроки с нормализацией индексов. Создай файлы `substring.js` и `substring.test.js`, следуй TDD.

**Функция принимает:**
- `str`: `string` — исходная строка
- `start`: `number` — начальный индекс (включая)
- `end`: `number` — конечный индекс (не включая)

**Функция возвращает:** `string` — извлечённая подстрока.

**Правила нормализации (как у стандартного `String.prototype.substring`):**
- Отрицательные и `NaN` → `0`
- Значения больше длины строки → длина строки
- Если `start > end` после нормализации — поменять их местами

**Напоминание по TDD + JSDoc:**
- Создай `substring.js` с заглушкой: `export function substring(str, start, end) { return ''; }`
- Сразу добавь JSDoc к заглушке
- Создай `substring.test.js` с первым тестом
- RED → GREEN → добавляй тесты по одному → REFACTOR

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна вернуть `'hello'` для `('hello', 0, 5)`
- Должна вернуть `'ell'` для `('hello', 1, 4)`
- Должна вернуть `'h'` для одного символа в начале `('hello', 0, 1)`
- Должна вернуть `'l'` для одного символа в середине `('hello', 2, 1)`
- Должна вернуть `'o'` для одного символа в конце `('hello', 4, 1)`
- Должна обрезать end до длины строки: `('hello', 0, 10)` → `'hello'`
- Должна вернуть до конца строки при не указанном 3 арг.: `('hello', 2)` → `'llo'`
- Должна поменять start и end местами если start > end: `('hello', 4, 1)` → `'ell'`
- Должна привести отрицательный start к 0: `('hello', -2, 3)` → `'hel'`
- Должна привести отрицательный end к 0, но start > 0: `('hello', 3, -1)` → `'hel'`
- Должна привести NaN к 0: `('hello', NaN, 3)` → `'hel'`
- Должна вернуть `''` если start >= длины строки: `('hello', 5, 7)` → `''`
- Должна вернуть `''` для пустой исходной строки: `('', 0, 1)` → `''`
- Должна работать с кириллицей: `('привет', 1, 4)` → `'рив'`
- Должна выбросить `TypeError` если str не строка
- Должна выбросить `TypeError` если start не указан или не число
- Должна выбросить `TypeError` если end не число
