# substring(str, start, end) — краткая выжимка

- **Извлечение подстроки** от start до end (не включая end), с нормализацией индексов
- **Нормализация:** отрицательные → 0, NaN → 0, > длины → длина строки
- **start > end** → меняются местами (в отличие от slice)
- `throw new TypeError` если аргументы не тех типов
