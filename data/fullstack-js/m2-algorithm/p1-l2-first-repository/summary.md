# Первый репозиторий — краткая выжимка

⏱ **Видео:** ~12 мин

- `git init` — создаёт репозиторий (папку `.git`) в текущей директории
- `git status` — показывает состояние: какие файлы изменены, какие не отслеживаются
- **Untracked** — файл есть в папке, но Git за ним не следит
- `.gitignore` — список файлов/папок, которые Git должен игнорировать (`node_modules/`, `.env`)
- Проект: `js-algorithms/` с `index.js`, `index.test.js` и `.gitignore`
