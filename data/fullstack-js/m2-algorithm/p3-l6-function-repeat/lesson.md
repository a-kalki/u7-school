# Функция repeat(str, count) — повторение строки

**Краткое содержание:**
Урок посвящён реализации функции повторения строки заданное количество раз. Студент самостоятельно пишет тесты и реализацию по текстовому описанию задачи и тестовых случаев.

### Алгоритм

Функция `repeat(str, count)` должна вернуть новую строку, состоящую из `str`, повторённой `count` раз.

**Шаги алгоритма:**
1. Проверить тип `str`: должен быть string. Если нет — `throw new TypeError`
2. Проверить тип `count`: должен быть number. Если нет — `throw new TypeError`
3. Проверить `count`: если `< 0` — `throw new RangeError('count must not be negative')`
4. Если `count === 0` — вернуть `''`
5. Создать пустую строку-результат `result = ''`
6. Повторить `count` раз: добавлять `str` к `result`
7. Вернуть `result`

**Обрати внимание:**
- `count` может быть не целым числом (например, `2.7`). Поведение: привести к целому через `Math.floor` или выбросить ошибку? В стандартном JS `String.prototype.repeat` обрезает дробную часть. Реализуй такое же поведение — используй `count = Math.floor(count)`.
- Проверку на `count < 0` нужно делать **после** `Math.floor`, иначе `-0.5` станет `-0` (и потом `0` — ок), а `-1.5` станет `-2` (ошибка).

### Реализация

Студент реализует функцию в файле `repeat.js` и тесты в `repeat.test.js`, следуя TDD-подходу (red → green → refactor).

**Напоминание:**
- Создай файл `repeat.js` с заглушкой: `export function repeat(str, count) { return ''; }`
- Создай файл `repeat.test.js`, импортируй `{ describe, test, expect } from 'bun:test'` и свою функцию
- Запусти `bun test` — тест должен упасть (RED)
- Реализуй функцию — тест должен пройти (GREEN)

### Тестовые случаи

Студент должен написать тесты для следующих случаев:

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Повтор 3 раза | `repeat('ab', 3)` | `'ababab'` |
| Повтор 1 раз | `repeat('hello', 1)` | `'hello'` |
| Повтор 0 раз | `repeat('abc', 0)` | `''` |
| Пустая строка | `repeat('', 5)` | `''` |
| Один символ | `repeat('x', 4)` | `'xxxx'` |
| Дробное count (2.7 → 2) | `repeat('a', 2.7)` | `'aa'` |
| Дробное count (0.5 → 0) | `repeat('a', 0.5)` | `''` |
| Кириллица | `repeat('да', 3)` | `'дадада'` |
| count отрицательный | `repeat('a', -1)` | `RangeError` |
| count не число | `repeat('a', '3')` | `TypeError` |
| str не строка | `repeat(123, 3)` | `TypeError` |
