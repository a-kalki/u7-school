---

### Реализация repeat()

**kind:** `text`

> Убедись, что ты в ветке `feat/substring-search`: `git switch feat/substring-search`

Реализуй функцию `repeat(str, count)` — повторение строки заданное количество раз. Создай файлы `repeat.js` и `repeat.test.js`, следуй TDD.

**Функция принимает:**
- `str`: `string` — строка для повторения
- `count`: `number` — количество повторений

**Функция возвращает:** `string` — строка, повторённая `count` раз.

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна повторить 3 раза: `repeat('ab', 3)` → `'ababab'`
- Должна вернуть исходную строку для `count = 1`: `repeat('hello', 1)` → `'hello'`
- Должна вернуть `''` для `count = 0`
- Должна вернуть `''` для пустой исходной строки: `repeat('', 5)` → `''`
- Должна повторить один символ: `repeat('x', 4)` → `'xxxx'`
- Должна обрезать дробную часть: `repeat('a', 2.7)` → `'aa'`
r Должна дать `''` для `count < 1`: `repeat('a', 0.5)` → `''`
r Должна дать вернуть `''` строку если `count` пропущен: `repeat('a')` → `''`
- Должна работать с кириллицей: `repeat('да', 3)` → `'дадада'`
- Должна выбросить `RangeError` для отрицательного `count`: `repeat('a', -1)`
- Должна выбросить `TypeError` если `count` не число: `repeat('a', '3')`
- Должна выбросить `TypeError` если `str` не строка: `repeat(123, 3)`
