# Что такое Git — краткая выжимка

⏱ **Видео:** ~15 мин

- **VCS (система контроля версий)** — инструмент для отслеживания изменений в коде
- **Репозиторий** — папка проекта под контролем Git (данные в `.git/`)
- **Коммит** — сохранённое состояние файлов; у каждого есть автор, сообщение, хеш
- **Stage (индекс)** — зона подготовки: выбираешь, что попадёт в следующий коммит
- **Ветка (branch)** — независимая линия разработки (основная — `main`)
- Установка: `sudo apt install git` (Linux) / `brew install git` (macOS) / скачать с сайта (Windows)
- Настройка: `git config --global user.name "..."` и `git config --global user.email "..."`
