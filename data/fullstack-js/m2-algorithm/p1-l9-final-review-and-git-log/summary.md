# Финальный прогон и git log — краткая выжимка

⏱ **Видео:** ~12 мин

- `bun test` — полный прогон всех 7 функций; все тесты должны быть зелёными
- `git log --oneline` — компактная история коммитов
- `git log -5` — последние N коммитов; `git log -p` — с diff-ом
- **Хороший commit-сообщение:** повелительное наклонение, до 72 символов, понятно без контекста
- Финальный коммит + домашнее задание: статья «Git для новичков (часть 1)» (Хабр), опционально — Unicode или TDD
