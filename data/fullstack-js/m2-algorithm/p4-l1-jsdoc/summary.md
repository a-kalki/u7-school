# JSDoc — краткая выжимка

⏱ **Видео:** ~15 мин

- **JSDoc** — стандарт документирования JS: комментарии `/** ... */` перед функцией
- **`@param {type} name — описание`** — параметры функции
- **`@returns {type} — описание`** — возвращаемое значение
- **`@throws {type} — описание`** — возможные исключения
- **`@example`** — пример использования
- **Типы:** `{string}`, `{number}`, `{boolean}`, `{any}`, `{string[]}`, `{object}`
- **Правило проекта:** каждая функция с этого момента — с JSDoc перед реализацией
- **VS Code** показывает JSDoc при наведении на вызов функции
