# Функция indexOf(str, search) — наивный поиск подстроки

**Краткое содержание:**
Урок посвящён реализации алгоритма поиска подстроки в строке — наивный (brute-force) поиск. Студент самостоятельно пишет тесты и реализацию по текстовому описанию задачи и тестовых случаев.

### Алгоритм

Функция `indexOf(str, search)` должна вернуть индекс первого вхождения подстроки `search` в строке `str`. Если подстрока не найдена — вернуть `-1`.

**Наивный алгоритм (brute-force):**
1. Проверить типы аргументов: оба должны быть string. Если нет — `throw new TypeError`
2. Если `search` — пустая строка (`''`), вернуть `0`
3. Если длина `search` больше длины `str` — вернуть `-1`
4. Пройти циклом `i` от 0 до `len(str) - len(search)`:
   - Для каждого `i` запустить вложенный цикл `j` от 0 до `len(search) - 1`
   - Сравнивать `str[i + j]` с `search[j]`
   - Если какой-то символ не совпал — прервать вложенный цикл (перейти к следующему `i`)
   - Если все символы совпали (вложенный цикл дошёл до конца) — вернуть `i`
5. Если после всех `i` совпадение не найдено — вернуть `-1`

**Пояснение алгоритма:**
Представь, что ты прикладываешь «окошко» (подстроку `search`) к строке и двигаешь его слева направо. На каждой позиции ты смотришь: совпадают ли все символы в «окошке» с символами строки? Если да — ты нашёл вхождение.

### Тестовые случаи

Студент должен написать тесты для следующих случаев:

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Подстрока в начале | `indexOf('hello', 'he')` | `0` |
| Подстрока в середине | `indexOf('hello', 'll')` | `2` |
| Подстрока в конце | `indexOf('hello', 'lo')` | `3` |
| Подстрока не найдена | `indexOf('hello', 'world')` | `-1` |
| Пустая поисковая строка | `indexOf('hello', '')` | `0` |
| Пустая исходная строка | `indexOf('', 'a')` | `-1` |
| Обе строки пустые | `indexOf('', '')` | `0` |
| Поисковая строка длиннее исходной | `indexOf('hi', 'hello')` | `-1` |
| Одинаковые строки | `indexOf('abc', 'abc')` | `0` |
| Несколько вхождений — первое | `indexOf('ababa', 'ba')` | `1` |
| Перекрытие: подстрока внутри почти-совпадения | `indexOf('abababc', 'ababc')` | `2` |
| Один символ | `indexOf('hello', 'h')` | `0` |
| Кириллица | `indexOf('привет', 'иве')` | `1` |
| Первый аргумент не строка | `indexOf(123, 'hello')` | `TypeError` |
| Второй аргумент не строка | `indexOf('hello', 123)` | `TypeError` |
