# indexOf(str, search) — краткая выжимка

- **Наивный поиск (brute-force):** «окошко» длиной с `search` движется по строке, на каждой позиции сравниваются символы
- **Алгоритм:** внешний цикл по `str`, вложенный по `search`; если все символы совпали — возвращаем позицию
- Если `search` — пустая строка → `0`
- Если `search` длиннее `str` → `-1` сразу (оптимизация)
- Несколько вхождений — возвращаем индекс **первого**
- `throw new TypeError` если аргументы не string
