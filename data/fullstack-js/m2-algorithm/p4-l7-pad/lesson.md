# Функция pad(str, length, char, side) — дополнение строки символами до заданной длины

**Краткое содержание:**
Урок посвящён реализации функции `pad(str, length, char, side)` — дополнение строки указанным символом до заданной длины слева, справа или с обеих сторон. Студент самостоятельно пишет тесты и реализацию.

### Алгоритм

Функция `pad(str, length, char, side)` должна дополнить строку `str` символами `char` до длины `length`. Параметр `side` определяет с какой стороны добавлять: `'left'`, `'right'` или `'both'`.

**Шаги алгоритма:**
1. Проверить типы: `str` и `char` — string, `length` — number, `side` — string. Если нет — `throw new TypeError`
2. Если `len(str) >= length` — вернуть `str` без изменений
3. Вычислить `padding = length - len(str)` — сколько символов нужно добавить
4. Собрать строку из `padding` символов `char` (через цикл)
5. В зависимости от `side`:
   - `'left'` — вернуть `paddingChars + str`
   - `'right'` — вернуть `str + paddingChars`
   - `'both'` — разделить `padding` пополам (`left = Math.floor(padding / 2)`, `right = padding - left`), вернуть `leftChars + str + rightChars`
   - Если `side` не `'left'`/`'right'`/`'both'` — выбросить ошибку

### Реализация

Студент реализует функцию в файле `pad.js` и тесты в `pad.test.js`, следуя TDD.

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Дополнение слева | `pad('hi', 5, '-', 'left')` | `'---hi'` |
| Дополнение справа | `pad('hi', 5, '-', 'right')` | `'hi---'` |
| Дополнение с обеих сторон (чётное) | `pad('hi', 6, '-', 'both')` | `'--hi--'` |
| Дополнение с обеих сторон (нечётное) | `pad('hi', 5, '-', 'both')` | `'--hi-'` |
| Строка уже нужной длины | `pad('hello', 5, '-', 'left')` | `'hello'` |
| Строка длиннее заданной длины | `pad('hello', 3, '-', 'left')` | `'hello'` |
| Дополнение пробелами | `pad('a', 3, ' ', 'right')` | `'a  '` |
| Дополнение многосимвольной строкой слева | `pad('hi', 7, 'ab', 'left')` | `'ababahi'` |
| Дополнение многосимвольной строкой справа | `pad('hi', 7, 'ab', 'right')` | `'hiababa'` |
| Дополнение пустым символом | решается студентом | - |
| Пустая исходная строка | `pad('', 4, 'x', 'both')` | `'xxxx'` |
| Длина = 0, пустая строка | `pad('', 0, '-', 'left')` | `''` |
| Неверный side | `pad('a', 3, '-', 'top')` | Ошибка |
| Аргумент не строка | `pad(123, 5, '-', 'left')` | `TypeError` |
