# Шаги урока: Функция pad()

---

### Реализация pad()

**kind:** `text`

Реализуй функцию `pad(str, length, char, side)` — дополнение строки символами до заданной длины.

**Функция принимает:**
- `str`: `string` — исходная строка
- `length`: `number` — целевая длина
- `char`: `string` — строка для дополнения (может быть длиннее одного символа; повторяется циклично и обрезается до нужной длины)
- `side`: `string` — сторона: `'left'`, `'right'` или `'both'`

**Правила для `'both'`:**
- Общее количество добавляемых символов: `padding = length - len(str)`
- Слева: `Math.floor(padding / 2)`, справа: `padding - left`
- При нечётном `padding` — лишний символ справа

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна дополнить слева: `('hi', 5, '-', 'left')` → `'---hi'`
- Должна дополнить справа: `('hi', 5, '-', 'right')` → `'hi---'`
- Должна дополнить с обеих сторон (чётное): `('hi', 6, '-', 'both')` → `'--hi--'`
- Должна дополнить с обеих сторон (нечётное, лишний справа): `('hi', 5, '-', 'both')` → `'-hi--'`
- Должна вернуть строку без изменений если длина уже достаточная: `('hello', 5, '-', 'left')` → `'hello'`
- Должна вернуть строку без изменений если она длиннее целевой: `('hello', 3, '-', 'left')` → `'hello'`
- Должна работать с пробелами: `('a', 3, ' ', 'right')` → `'a  '`
- Должна работать с пустой исходной строкой: `('', 4, 'x', 'both')` → `'xxxx'`
- Должна вернуть `''` для `pad('', 0, '-', 'left')`
- Должна дополнить многосимвольной строкой слева: `('hi', 7, 'ab', 'left')` → `'ababahi'` (padding=5, 'ab' повторяется и обрезается)
- Должна дополнить многосимвольной строкой справа: `('hi', 7, 'ab', 'right')` → `'hiababa'`
- Должна выбросить ошибку при неверном side: `('a', 3, '-', 'top')`
- Должна выбросить `TypeError` если str не строка
