---

### Шаг 1 — Финальная проверка

**kind:** `text`

Перед мержем убедись, что все условия выполнены:

- [ ] Получено 2 аппрува (зелёные галочки рядом с именами ревьюеров)
- [ ] Все conversation помечены как resolved
- [ ] Нет незакрытых комментариев
- [ ] `bun test` проходит локально (на всякий случай)

Если что-то не выполнено — доведи до конца.

---

### Шаг 2 — Мерж через интерфейс GitHub

**kind:** `text`

Выполни мерж PR:

1. Открой свой PR на GitHub
2. Нажми зелёную кнопку **Merge pull request**
3. Оставь тип слияния **Merge commit** (по умолчанию)
4. Нажми **Confirm merge**
5. После успешного мержа нажми **Delete branch** — удалить ветку на GitHub

---

### Шаг 3 — Обновление локального main

**kind:** `text`

Подтяни изменения в локальный репозиторий:

```
git checkout main
git pull origin main
```

Проверь, что коммиты из `feat/substring-search` теперь в `main`:

```
git log --oneline
```

---

### Шаг 4 — Очистка локальных веток

**kind:** `text`

Удали локальную feature-ветку:

```
git branch -d feat/substring-search
```

Очисти информацию об удалённых ветках:

```
git fetch --prune
```

Проверь, что остались только нужные ветки:

```
git branch -a
```
