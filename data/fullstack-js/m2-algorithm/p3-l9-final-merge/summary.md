# Финальный мерж — краткая выжимка

- **Перед мержем:** 2 аппрува, все conversation resolved
- **Мерж на GitHub:** Merge pull request → Confirm merge → Delete branch
- **Локально:** `git checkout main` → `git pull origin main`
- **Очистка:** `git branch -d feat/substring-search`, `git fetch --prune`
- **Типы слияния:** merge commit (история сохраняется), squash (всё в один коммит), rebase (линейная история)
- **Итог:** полный цикл Git + 6 функций поиска и утилит для строк
