# TDD и функция len() — краткая выжимка

⏱ **Видео:** ~22 мин

- **TDD** — разработка через тестирование: RED (тест падает) → GREEN (минимальный код) → REFACTOR (улучшение)
- **bun:test** — встроенный тестовый фреймворк: `describe`, `test`, `expect(x).toBe(y)`
- **Шаг 1:** создай `len.js` с заглушкой `return ''`, экспортируй через `export function`
- **Шаг 2:** создай `len.test.js`, импортируй `{ describe, test, expect } from 'bun:test'` и `{ len } from './len.js'`
- **Шаг 3:** запусти `bun test` — тест должен упасть (RED)
- **Шаг 4:** замени заглушку на подсчёт символов циклом до `undefined` (GREEN)
- **Шаг 5:** добавь тесты на граничные случаи (пустая строка, пробелы, кириллица, исключения)
- `expect(() => fn()).toThrow(TypeError)` — проверка, что функция выбрасывает ошибку
- `git add` + `git commit -m "..."` — сохраняем пройденный тест
