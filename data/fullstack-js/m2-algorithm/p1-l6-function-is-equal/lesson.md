# Функция isEqual(a, b) — посимвольное сравнение строк

**Краткое содержание:**
Урок посвящён реализации функции `isEqual(a, b)` — посимвольное сравнение двух строк на равенство. Студент самостоятельно пишет тесты и реализацию по текстовому описанию задачи и тестовых случаев.

### Алгоритм

Функция `isEqual(a, b)` должна вернуть `true`, если строки посимвольно равны, и `false` в противном случае.

**Шаги алгоритма:**
1. Проверить типы аргументов: оба должны быть string. Если нет — `throw new TypeError`
2. Сравнить длины строк через `len()`. Если длины разные — строки точно не равны, вернуть `false`
3. Пройти циклом от 0 до длины, сравнивая `a[i]` и `b[i]`. Если на каком-то шаге символы не равны — вернуть `false`
4. Если цикл завершился и все символы совпали — вернуть `true`

### Реализация

Студент реализует функцию в файле `isEqual.js` и тесты в `isEqual.test.js`, следуя TDD-подходу (red → green → refactor).

**Напоминание:**
- Создай файл `isEqual.js` с заглушкой: `export function isEqual(a, b) { return false; }`
- Создай файл `isEqual.test.js`, импортируй `{ describe, test, expect } from 'bun:test'` и свою функцию
- Запусти `bun test` — тест должен упасть (RED)
- Реализуй функцию — тест должен пройти (GREEN)

### Тестовые случаи

Студент должен написать тесты для следующих случаев:

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Равные строки | `isEqual('hello', 'hello')` | `true` |
| Равные пустые строки | `isEqual('', '')` | `true` |
| Разные строки одинаковой длины | `isEqual('hello', 'world')` | `false` |
| Строки разной длины | `isEqual('hi', 'hello')` | `false` |
| Одна строка пустая, другая нет | `isEqual('', 'a')` | `false` |
| Пробелы имеют значение | `isEqual(' a', 'a')` | `false` |
| Кириллица | `isEqual('привет', 'привет')` | `true` |
| Первый аргумент не строка | `isEqual(123, 'hello')` | `TypeError` |
| Второй аргумент не строка | `isEqual('hello', null)` | `TypeError` |
| Оба аргумента не строки | `isEqual(1, 2)` | `TypeError` |
