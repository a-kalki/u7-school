# Функция lowerCase(str) — перевод в нижний регистр через коды символов

**Краткое содержание:**
Урок посвящён реализации функции `lowerCase(str)` — перевод заглавных букв в строчные через коды символов. Зеркальная функция к `upperCase`: работает как для латиницы (`'A'`–`'Z'` → `'a'`–`'z'`), так и для кириллицы (`'А'`–`'Я'` → `'а'`–`'я'`). В обоих случаях прибавляется 32.

### Алгоритм

Функция `lowerCase(str)` должна вернуть новую строку, в которой все заглавные буквы (латиница и кириллица) заменены на строчные. Остальные символы остаются без изменений.

**Шаги алгоритма:**
1. Проверить тип: `str` должен быть string. Если нет — `throw new TypeError`
2. Создать пустую строку-результат
3. Пройти циклом по каждому символу `str[i]`:
   - Получить код символа через `.charCodeAt(0)`
   - Если код в диапазоне 65–90 (заглавная латиница `'A'`–`'Z'`) **или** 1040–1071 (заглавная кириллица `'А'`–`'Я'`):
     - Прибавить 32 (65 → 97 = `'a'`, 1040 → 1072 = `'а'`)
     - Преобразовать код обратно в символ через `String.fromCodePoint(код)`
     - Добавить символ к результату
   - Иначе — добавить исходный символ к результату
4. Вернуть результат

### Тестовые случаи

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Все заглавные латинские | `lowerCase('HELLO')` | `'hello'` |
| Уже строчные | `lowerCase('hello')` | `'hello'` |
| Смешанный регистр | `lowerCase('Hello World')` | `'hello world'` |
| Строка с цифрами | `lowerCase('HELLO123')` | `'hello123'` |
| Строка со знаками | `lowerCase('HELLO!@#')` | `'hello!@#'` |
| Пустая строка | `lowerCase('')` | `''` |
| Только не-буквы | `lowerCase('123!@#')` | `'123!@#'` |
| Кириллица | `lowerCase('ПРИВЕТ')` | `'привет'` |
| Смесь латиницы и кириллицы | `lowerCase('HELLO ПРИВЕТ')` | `'hello привет'` |
| Уже строчная кириллица | `lowerCase('привет')` | `'привет'` |
| Аргумент не строка | `lowerCase(123)` | `TypeError` |
