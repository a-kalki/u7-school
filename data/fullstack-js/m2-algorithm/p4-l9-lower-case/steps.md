# Шаги урока: Функция lowerCase()

---

### Реализация lowerCase()

**kind:** `text`

Реализуй функцию `lowerCase(str)` — перевод заглавных букв в строчные через коды символов. Работает как для латиницы, так и для кириллицы.

**Функция принимает:**
- `str`: `string` — исходная строка

**Функция возвращает:** `string` — строка, в которой все заглавные буквы (`'A'`–`'Z'`, `'А'`–`'Я'`) заменены на соответствующие строчные. Остальные символы не меняются.

Тестовые случаи (каждый — отдельный `test(...)`):
- Должна перевести все заглавные латинские: `('HELLO')` → `'hello'`
- Должна оставить строчные без изменений: `('hello')` → `'hello'`
- Должна работать со смешанным регистром: `('Hello World')` → `'hello world'`
- Должна не трогать пробелы: `(' HELLO 123')` → `' hello 123'`
- Должна не менять цифры: `('HELLO123')` → `'hello123'`
- Должна не менять знаки: `('HELLO!@#')` → `'hello!@#'`
- Должна вернуть `''` для пустой строки
- Должна вернуть строку без изменений если нет заглавных букв: `('123!@#')` → `'123!@#'`
- Должна перевести кириллицу: `('ПРИВЕТ')` → `'привет'`
- Должна работать со смесью латиницы и кириллицы: `('HELLO ПРИВЕТ')` → `'hello привет'`
- Должна не менять уже строчные буквы (включая кириллицу): `('привет')` → `'привет'`
- Должна выбросить `TypeError` если аргумент не строка
