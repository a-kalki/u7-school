# Функции includes(), startsWith(), endsWith() — проверки наличия подстроки

**Краткое содержание:**
Урок посвящён реализации трёх функций для проверки наличия подстроки. Студент самостоятельно пишет тесты и реализации по текстовому описанию задач и тестовых случаев.

### 1. Функция includes(str, search)

Функция проверяет, содержится ли подстрока `search` в строке `str`. Это обёртка над `indexOf`: вернуть `true` если `indexOf(str, search) !== -1`, и `false` в противном случае.

### 2. Функция startsWith(str, search)

Функция проверяет, начинается ли строка `str` с подстроки `search`. Алгоритм:
1. Проверить типы аргументов
2. Если `search` — пустая строка, вернуть `true`
3. Если длина `search` больше длины `str` — вернуть `false`
4. Пройти циклом по символам `search` и сравнить с соответствующими символами `str` (индексы 0, 1, 2...)
5. Если все совпали — `true`, иначе `false`

### 3. Функция endsWith(str, search)

Функция проверяет, заканчивается ли строка `str` подстрокой `search`. Алгоритм аналогичен `startsWith`, но сравниваются последние символы `str`:
1. Проверить типы аргументов
2. Если `search` — пустая строка, вернуть `true`
3. Если длина `search` больше длины `str` — вернуть `false`
4. Вычислить смещение: `offset = len(str) - len(search)`
5. Пройти циклом по символам `search` и сравнить с `str[offset + i]`
6. Если все совпали — `true`, иначе `false`

### Реализация

Студент реализует функции в файлах `includes.js`, `starts-with.js`, `ends-with.js` и тесты в `includes.test.js`, `starts-with.test.js`, `ends-with.test.js`, следуя TDD-подходу.

**Напоминание:**
- Для `includes` импортируй `indexOf` из `index-of.js`: `import { indexOf } from './index-of.js'`
- Для каждой функции — отдельный файл с заглушкой и отдельный файл тестов

### Тестовые случаи для includes()

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Подстрока найдена | `includes('hello', 'll')` | `true` |
| Подстрока не найдена | `includes('hello', 'world')` | `false` |
| Пустая поисковая строка | `includes('hello', '')` | `true` |
| Поисковая строка длиннее исходной | `includes('hi', 'hello')` | `false` |
| Перекрытие: подстрока внутри почти-совпадения | `includes('abababc', 'ababc')` | `true` |
| Первый аргумент не строка | `includes(123, 'a')` | `TypeError` |

### Тестовые случаи для startsWith()

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Строка начинается с подстроки | `startsWith('hello', 'hel')` | `true` |
| Строка не начинается с подстроки | `startsWith('hello', 'llo')` | `false` |
| Пустая поисковая строка | `startsWith('hello', '')` | `true` |
| Поисковая строка длиннее исходной | `startsWith('hi', 'hello')` | `false` |
| Точное совпадение | `startsWith('abc', 'abc')` | `true` |
| Первый аргумент не строка | `startsWith(123, 'a')` | `TypeError` |

### Тестовые случаи для endsWith()

| Случай | Входные данные | Ожидаемый результат |
|--------|---------------|---------------------|
| Строка заканчивается подстрокой | `endsWith('hello', 'llo')` | `true` |
| Строка не заканчивается подстрокой | `endsWith('hello', 'hel')` | `false` |
| Пустая поисковая строка | `endsWith('hello', '')` | `true` |
| Поисковая строка длиннее исходной | `endsWith('hi', 'hello')` | `false` |
| Точное совпадение | `endsWith('abc', 'abc')` | `true` |
| Один символ в конце | `endsWith('hello', 'o')` | `true` |
| Первый аргумент не строка | `endsWith(123, 'a')` | `TypeError` |
