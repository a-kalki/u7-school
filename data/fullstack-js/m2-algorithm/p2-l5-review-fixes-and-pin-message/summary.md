# Доработка по ревью и pin-сообщение — краткая выжимка

⏱ **Видео:** ~10 мин

- **Ответ на замечание:** «Согласен, исправил» ИЛИ «Не согласен, потому что...»
- **Пуш правок** → PR обновляется автоматически (не нужно создавать новый)
- **Resolve conversation** — отметить комментарий как решённый после исправления
- **Re-request review** (🔄) — запросить повторную проверку
- **Pin-сообщение:** ссылка на PR + статус в закреплённом сообщении чата
- **Ревьюер не отвечает:** напомнить → пингануть → найти другого → эскалировать
- **Домашнее задание:** статья «Git для новичков (часть 2)» — командная работа: ветки, merge, clone
