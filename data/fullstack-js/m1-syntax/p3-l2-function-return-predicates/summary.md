# Функция: return, предикаты и методы реализации — краткая выжимка

⏱ **Видео:** 20:00

- **`return`** немедленно завершает функцию; можно использовать для досрочного выхода при валидации
- **Без явного `return`** функция неявно возвращает `undefined`
- **Предикаты** — функции-вопросы: `isString()`, `hasChildren()`, `canKiss()`; всегда возвращают boolean
- **Задача vs Реализация:** *что* делает функция (цель) ≠ *как* делает (код); одна задача — много реализаций
- **Рекурсия** — альтернативный способ реализации: функция вызывает саму себя
