# Типы данных, исключения и особые случаи — краткая выжимка

⏱ **Видео:** 10:51

- **Оператор `+`** со строками выполняет конкатенацию, с числами — сложение (неявное приведение типов)
- **Арифметика со строками:** `'5' / 5` → `1` (строка приводится к числу), `'S' / 5` → `NaN`
- **Методы строк** (например, `.toUpperCase()`) доступны только для строкового типа
- **Исключение (TypeError)** возникает при вызове метода несовместимого типа и останавливает весь дальнейший код
- **Важно понимать типы данных,** чтобы писать корректный код — интерпретатор ведёт себя по-разному в зависимости от типа
