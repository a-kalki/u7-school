# Функции: тип, свойства и стили объявления — краткая выжимка

⏱ **Видео:** 11:55

- **`typeof` функции** → `'function'`; функция — объект, может иметь собственные свойства
- **Три стиля объявления:** Function Declaration (именованная, хойстится), Function Expression (анонимная, не хойстится), Arrow Function (краткий синтаксис)
- **First-class citizens:** функции можно присваивать переменным, передавать как аргументы, возвращать
- **Стрелочные функции:** `() => expr` — краткий синтаксис; без `{}` — неявный `return`
- **`sayHi.call()`** — функции внутри используют метод `call`; можно добавлять свойства вроде `sayHi.callCount`
