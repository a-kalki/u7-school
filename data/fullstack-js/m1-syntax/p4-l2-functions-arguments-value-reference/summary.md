# Функции: аргументы, передача по значению и ссылке — краткая выжимка

⏱ **Видео:** 13:52

- **Параметры ≠ Аргументы:** параметры в объявлении, аргументы при вызове
- **Пропущенный аргумент** → `undefined`; ошибки не будет, но может сломаться логика
- **Простые типы (числа, строки, boolean):** передаются по значению — копируются, оригинал не меняется
- **Сложные типы (объекты, массивы):** передаются по ссылке — изменения внутри функции видны снаружи
- **Критично понимать** разницу для предсказуемого поведения кода
