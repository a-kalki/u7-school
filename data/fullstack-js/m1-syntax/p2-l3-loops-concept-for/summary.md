# Циклы: концепция и основы for — краткая выжимка

⏱ **Видео:** 13:21

- **Цикл** — многократное выполнение блока кода до достижения условия
- **Синтаксис `for`:** `for (инициализация; условие; обновление) { тело }`
- **Поток выполнения:** инициализация → проверка условия → тело → обновление → проверка условия → ...
- **`for` удобен, когда количество итераций известно заранее**
- **Все три части (`init`, `cond`, `expr`) объединены в одной строке**
