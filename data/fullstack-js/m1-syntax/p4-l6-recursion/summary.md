# Рекурсия — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Рекурсия** — функция вызывает саму себя; состоит из базы (условие выхода) и шага (упрощение задачи)
- **Контекст выполнения** — каждый вызов создаёт новый контекст в стеке; глубина ограничена (~10000)
- **Рекурсивный обход** — идеален для вложенных структур неизвестной глубины (DOM, JSON, дерево отделов)
- **Связный список** — рекурсивная структура данных: `{ value, next }`; быстрая вставка/удаление
- **Итеративно — экономичнее по памяти,** рекурсивно — короче и нагляднее; любая рекурсия переписывается в цикл
