# Переменные — краткая выжимка

⏱ **Видео:** 12:59

- **Консоль (F12)** — инструмент взаимодействия с интерпретатором JS в браузере для тестирования и отладки
- **Переменная** — именованная область в heap (куче), существует только во время выполнения программы
- **Три оператора создания:** `let` (изменяемая), `const` (неизменяемая), `var` (устаревший, не использовать)
- **Оператор присваивания `=`** записывает значение в переменную; `undefined` — значение неприсвоенной переменной
- **Всегда предпочитайте `const`**, если значение не планируется менять — это снижает вероятность ошибок
