# Структура данных: Map и Set — краткая выжимка

⏱ **Чтение:** ~20 мин

- **Map** — коллекция пар ключ-значение с любым типом ключа; методы: `set`, `get`, `has`, `delete`, `size`
- **Set** — коллекция уникальных значений; методы: `add`, `has`, `delete`, `size`; `has` за O(1)
- **Главное отличие Object от Map:** в Map ключом может быть объект, порядок вставки сохраняется
- **Set vs Array:** Set не допускает дубликаты, `has` быстрее `indexOf`
- **Удаление дубликатов:** `[...new Set(arr)]`
