# Введение в программирование — краткая выжимка

⏱ **Видео:** 11:11

- **Код** — инструкция на понятном компьютеру языке; программа — множество кода, объединённого для решения большой задачи
- **Интерпретатор** (например, V8 в Chrome) читает и выполняет код построчно
- **Поток выполнения (thread)** — последовательное исполнение кода сверху вниз, с возможностью переходов
- **JavaScript** — динамический, востребованный язык для фронтенда и бэкенда; сложен для больших проектов (решается TypeScript)
- **Программист** — решает проблемы пользователей через написание хорошего и рабочего кода
