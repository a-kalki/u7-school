# Циклы: for для перебора строк — краткая выжимка

⏱ **Видео:** 10:38

- **Строка — коллекция символов** с доступом по индексу: `str[0]`, `str[1]`, ...
- **Свойство `.length`** возвращает количество символов; используется как верхняя граница цикла
- **Стандартный паттерн:** `for (let i = 0; i < str.length; i++) { str[i] }`
- **Индексация с нуля:** первый символ — `[0]`, последний — `[str.length - 1]`
- **Применение:** посимвольный анализ, трансформация, поиск в строке
