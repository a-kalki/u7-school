# Логические операторы — краткая выжимка

⏱ **Видео:** 16:48

- **ИЛИ `||`** — возвращает `true`, если хоть один операнд истинен; **И `&&`** — если оба истинны
- **Таблицы истинности:** `true || false` → `true`; `true && false` → `false`
- **Сложные выражения комбинируют `&&` и `||`**; используйте скобки для явного порядка и читаемости
- **Практическое применение:** проверка диапазонов, условий трудоспособности, валидация данных
- **Логические операторы возвращают не только `true`/`false`** — подробнее в уроке 12
