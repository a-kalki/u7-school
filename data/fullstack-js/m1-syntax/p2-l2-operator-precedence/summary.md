# Приоритеты выполнения операторов — краткая выжимка

⏱ **Видео:** 17:08

- **Приоритет (от высшего к низшему):** `()` → унарные `! + -` → `**` → `* / %` → `+ -` → сравнения → равенства → `&&` → `||/??` → тернарный `? :` → присваивание `=`
- **Скобки `()` — высший приоритет,** используйте их для явного управления порядком
- **Присваивание — самый низкий приоритет,** вычисляется справа налево
- **Ассоциативность:** большинство операторов — слева направо; `**` и присваивания — справа налево
- **Рекомендация:** всегда используйте скобки или разбивайте сложные выражения на отдельные строки
