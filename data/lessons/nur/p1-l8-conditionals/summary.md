# Ветвление — краткая выжимка

⏱ **Видео:** 21:07

- **`if (условие)`** — выполняет блок кода, если условие истинно; условие должно быть булевым
- **`else if`** — проверяет дополнительное условие, если предыдущее ложно; можно использовать многократно
- **`else`** — выполняется, если все условия ложны; не имеет своего условия
- **В цепочке if-else if-else выполняется только первый истинный блок**
- **Однострочный синтаксис без `{}`** допустим для одной инструкции, но `{}` улучшают читаемость
