# Анонимные и стрелочные функции — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Анонимная функция** — функция без имени, создаётся как Function Expression: `let fn = function() {}`
- **Стрелочная функция:** `(args) => expr` — краткий синтаксис; без `{}` — неявный return
- **Вариации:** один аргумент — без скобок `n => n*2`; нет аргументов — `() => ...`; тело — `{}` + `return`
- **Отличия от `function`:** нет своего `this`, нет `arguments`, нельзя `new`, не хойстятся
- **Когда использовать:** короткие вычисления, колбэки, когда нужно сохранить внешний `this`
