# Операторы равенства сравнения и отрицания

```markdown
# Основы JavaScript: Операторы равенства, сравнения и отрицания

## Краткое содержание
В этом видео-уроке рассматриваются основные операторы JavaScript, используемые для сравнения значений: операторы равенства и неравенства (строгие и нестрогие), операторы сравнения (больше, меньше, больше или равно, меньше или равно) и оператор отрицания. Особое внимание уделяется различиям между строгими и нестрогими операторами равенства и даются рекомендации по их использованию.

## Основные темы
1.  Операторы равенства и неравенства
2.  Операторы сравнения
3.  Оператор отрицания (инвертирования)

### Операторы равенства и неравенства

**Концепция:**
Операторы равенства и неравенства используются для определения, равны ли два значения между собой или нет. В JavaScript существуют строгие и нестрогие варианты этих операторов.

*   **Строгое равенство (`===`):** Проверяет, равны ли значения И их типы данных.
*   **Нестрогое равенство (`==`):** Проверяет, равны ли значения, при этом JavaScript может выполнять преобразование типов (coercion) перед сравнением.
*   **Строгое неравенство (`!==`):** Проверяет, не равны ли значения И/ИЛИ их типы данных.
*   **Нестрогое неравенство (`!=`):** Проверяет, не равны ли значения, с возможным преобразованием типов.

**Примеры кода (из консоли и слайда):**

```javascript
// Строгое равенство
5 === 5;    // -> true (значения и типы совпадают)

// Строгое неравенство
5 !== 5;    // -> false (значения и типы совпадают)

// Сравнение числа и строки
'5' === 5;  // -> false (значения равны, но типы разные)

// Нестрогое равенство
'5' == 5;   // -> true (JavaScript преобразует строку '5' в число 5, затем сравнивает)
```

**Примеры с переменными (из слайда):**

```javascript
let i = 5;
i === 5;    // -> true
i == 5;     // -> true
i !== 5;    // -> false
i != 5;     // -> false

let j = 5;
j === 3;    // -> false
j == 3;     // -> false
j !== 3;    // -> true
j != 3;     // -> true
```

**Важные замечания и нюансы:**
*   Опытные программисты предпочитают **не использовать** операторы нестрогого равенства (`==`) и нестрогого неравенства (`!=`).
*   Причина в том, что нестрогие операторы могут приводить к **труднонаходимым ошибкам** из-за неявного преобразования типов.
*   Всегда используйте строгие операторы (`===` и `!==`), если нет конкретной причины для обратного.

---

### Операторы сравнения

**Концепция:**
Операторы сравнения используются, когда необходимо сравнить два значения и определить их относительный порядок (больше, меньше и т.д.).

**Типы операторов сравнения:**
*   `<`: меньше
*   `>`: больше
*   `<=`: меньше или равно
*   `>=`: больше или равно

**Примеры кода (из консоли и слайда):**

```javascript
// Меньше
5 < 7;      // -> true

let bar = 5;
bar < 0;    // -> false (5 не меньше 0)
```

**Примеры с переменными (из слайда):**

```javascript
let i = 5;
i > 3;      // -> true
i < 3;      // -> false
i <= 3;     // -> false (5 не меньше или равно 3)
i >= 5;     // -> true (5 больше или равно 5)

let j = 5;
j > 10;     // -> false
j < 10;     // -> true
j >= 10;    // -> false
j <= 10;    // -> true (5 меньше или равно 10)
```

---

### Оператор отрицания (инвертирования) `НЕ`

**Концепция:**
Оператор отрицания (`!`) используется для инвертирования булева значения. Если значение `true`, он делает его `false`, и наоборот.

**Синтаксис и действие:**
Оператор отрицания `!` "переворачивает" булево значение в обратное.
*   Не истина (`!true`) === ложь (`false`)
*   Не ложь (`!false`) === истина (`true`)

**Примеры кода (из слайда):**

```javascript
!true;  // -> false
!false; // -> true

let i = 5;
!(i === 5); // -> false (i === 5 возвращает true, !true возвращает false)
i !== 5;    // -> false (эквивалентно предыдущему выражению)
```

**Пример из жизни:**
*   Если (сегодня выходной день), то остаюсь дома.
*   Если (сегодня **НЕ** выходной день), то иду на работу.
    *   Здесь "НЕ выходной день" эквивалентно "рабочий день".

---

## Ключевые выводы
1.  Все операторы равенства, неравенства, сравнения и отрицания **ВСЕГДА возвращают булевый тип:** `true` или `false`.
2.  Строгие операторы равенства (`===`) и неравенства (`!==`) сравнивают как значение, так и тип данных.
3.  Нестрогие операторы равенства (`==`) и неравенства (`!=`) могут выполнять неявное преобразование типов, что часто приводит к непредсказуемому поведению и ошибкам.
4.  Рекомендуется всегда использовать строгие операторы (`===`, `!==`) для предотвращения ошибок, связанных с преобразованием типов.
5.  Оператор отрицания (`!`) инвертирует булево значение выражения, к которому он применяется.

## Термины
*   **Операторы равенства/неравенства:** Символы, используемые для проверки, равны ли два значения друг другу (или не равны).
*   **Строгое равенство (`===`):** Проверка на равенство, учитывающая как значение, так и тип данных.
*   **Нестрогое равенство (`==`):** Проверка на равенство, которая может выполнять преобразование типов перед сравнением.
*   **Операторы сравнения:** Символы (`<`, `>`, `<=`, `>=`), используемые для сравнения относительного размера или порядка двух значений.
*   **Оператор отрицания (`!`):** Символ, который инвертирует булево значение (true становится false, false становится true).
*   **Булевый тип (`boolean`):** Тип данных, который может принимать только два значения: `true` (истина) или `false` (ложь).
*   **Преобразование типов (Type Coercion):** Автоматическое или неявное преобразование типов данных в JavaScript, часто происходящее при использовании нестрогих операторов.
