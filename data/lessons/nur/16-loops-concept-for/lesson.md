# Циклы: концепция и основы for

### Часть 1 (00:00-13:21): Концепция циклов и основы for

**Концепция циклов:**
*   Циклы в программировании — это механизм для многократного выполнения одного и того же блока кода.
*   Они используются, когда нужно повторять действия до выполнения определенного условия (например, "мыть яблоки, пока ящик не опустеет").

**Оператор `for`:**
*   Наиболее часто используемая циклическая конструкция.
*   **Синтаксис:**
    ```javascript
    for ({initExpr}; {condBeforeCycle}; {exprAfterCycle}) {
        cycleBodyCode;
    }
    ```
    *   `initExpr` (инициализация): Выполняется **один раз** в начале цикла (например, `let i = 0;`).
    *   `condBeforeCycle` (условие): Проверяется **перед каждой итерацией**. Если `true`, цикл продолжается; если `false`, цикл завершается.
    *   `exprAfterCycle` (изменение): Выполняется **после каждой итерации** тела цикла (например, `i += 1;`).
    *   `cycleBodyCode` (тело цикла): Код, который выполняется многократно.
*   **Поток выполнения:** `initExpr` -> (проверка `condBeforeCycle` -> выполнение `cycleBodyCode` -> выполнение `exprAfterCycle`) -> повтор проверки условия.

**Пример первого цикла `for`:**
```javascript
for (let i = 0; i < 3; i += 1) {
    console.log(i + ': Hello world');
}
```
*   **Разбор:**
    *   `let i = 0`: Инициализация счетчика `i`.
    *   `i < 3`: Условие продолжения.
    *   `i += 1`: Увеличение `i` на 1 после каждой итерации.
*   **Вывод в консоль:**
    ```
    0: Hello world
    1: Hello world
    2: Hello world
    ```

**Ключевые выводы:**
*   Циклы автоматизируют повторяющиеся действия.
*   Оператор `for` удобен, когда количество итераций известно заранее.
*   `for` объединяет инициализацию, условие и обновление счетчика в одной строке.
