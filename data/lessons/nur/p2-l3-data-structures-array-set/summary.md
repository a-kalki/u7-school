# Структура данных: Array (массив) — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Array** — упорядоченная коллекция с числовыми индексами; создаётся через `[]`
- **Методы:** `push`/`pop` (конец, быстро), `shift`/`unshift` (начало, медленно); `length`, `at(-1)`
- **Перебор:** `for` по индексу или `for..of` по значениям
- **Стек (LIFO):** push + pop; **Очередь (FIFO):** push + shift
