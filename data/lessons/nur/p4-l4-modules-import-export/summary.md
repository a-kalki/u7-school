# Краткое содержание

- Модуль — файл с изолированной областью видимости, всё приватно пока не `export`
- Named export: `export function/const/class`, импорт через `import { имя }`
- Default export: `export default`, импорт без скобок, имя выбирает импортёр
- Реэкспорт: barrel-файлы (`index.js`) собирают экспорты из нескольких модулей
- Порядок загрузки: зависимости разрешаются рекурсивно, модуль выполняется один раз и кешируется
- Bun: модули ES по умолчанию, импорты работают «из коробки»
