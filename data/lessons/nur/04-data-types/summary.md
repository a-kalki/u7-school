# Типы данных — краткая выжимка

⏱ **Видео:** 19:06

- **Основные типы JS:** `number` (все числа, включая NaN и Infinity), `string`, `boolean`, `object`, `null`, `undefined`
- **Оператор `typeof`** — возвращает строку с типом значения в runtime; `typeof null` → `"object"` (особенность JS)
- **Строки** — в одинарных, двойных или обратных кавычках; обратные кавычки поддерживают интерполяцию `${}`
- **`undefined`** — переменная объявлена, но не присвоена; **`null`** — намеренное «ничего» (лучше избегать)
- **Исключение (exception)** — ошибка типа (TypeError), которая останавливает поток выполнения
