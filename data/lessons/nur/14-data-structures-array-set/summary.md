# Структура данных: последовательность (Array, Set) — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Array** — упорядоченная коллекция с числовыми индексами; создаётся через `[]`
- **Методы:** `push`/`pop` (конец, быстро), `shift`/`unshift` (начало, медленно); `length`, `at(-1)`
- **Set** — коллекция уникальных значений без ключей; `add`, `delete`, `has`, `size`; перебор через `for..of`
- **Главное отличие:** в Set дубликаты автоматически удаляются; `has` работает за O(1)
- **Типичное применение Set:** удаление дубликатов — `[...new Set(arr)]`
