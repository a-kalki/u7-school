# Циклы: for для перебора строк

### Часть 2 (13:21-23:59): For для перебора строк

**Цикл `for` для перебора символов в строке:**
*   `for` часто используется для доступа к каждому элементу коллекции по индексу. Строки в JavaScript — это коллекции символов.
*   **Свойство `.length`:** У любой строки есть свойство `length`, возвращающее количество символов.
    ```javascript
    let s = "string";
    console.log(s.length); // 6
    ```
*   **Доступ к символам по индексу:** Символы в строке доступны по числовому индексу, который начинается с `0`.
    ```javascript
    let word = "hello";
    console.log(word[0]); // 'h'
    console.log(word[4]); // 'o'
    ```

**Пример перебора строки:**
```javascript
let h = 'programmer';
console.log('length: ' + h.length);

for (let i = 0; i < h.length; i += 1) {
    const s = i + ': ' + h[i]; // Получаем символ по индексу i
    console.log(s);
}
```
*   **Вывод в консоль:**
    ```
    length: 10
    0: p
    1: r
    2: o
    3: g
    4: r
    5: a
    6: m
    7: m
    8: e
    9: r
    ```

**Ключевые выводы:**
*   `for` отлично подходит для итерации по строкам с использованием индексов.
*   Свойство `.length` строки определяет верхнюю границу для счетчика цикла.
*   Доступ к `string[index]` позволяет получить отдельный символ.
