# Явное приведение типов

**Краткое содержание:**
Данный урок посвящен понятию "приведение типов" в JavaScript. Мы разберем, что это такое, чем отличается явное приведение типов от неявного, и подробно рассмотрим различные способы явного преобразования данных между типами `string`, `number`, `boolean`, а также поведение объектов, `null` и `undefined` при приведении к булевому типу.

**Основные темы:**
1.  Определение приведения типов.
2.  Отличия явного и неявного приведения типов.
3.  Явное приведение `number` и `boolean` к `string`.
4.  Явное приведение `string` к `number`.
5.  Явное приведение `string`, `number`, `object`, `undefined`, `null` к `boolean`.
6.  Сводка "ложных" (falsy) значений в JavaScript.

---

### Подробный конспект

#### 1. Что такое приведение типов?

**Объяснение концепции:**
Приведение типов (Type Coercion) — это процесс изменения типа данных в JavaScript во время выполнения кода. Это может происходить автоматически (неявное приведение) или по явному указанию разработчика (явное приведение).

**Отличия явного и неявного приведения типов:**

*   **Неявное приведение типов:** Происходит, когда в коде явно не указано изменение типа, но интерпретатор JavaScript выполняет его самостоятельно, согласно внутренним правилам языка.
*   **Явное приведение типов:** Происходит, когда разработчик в коде дает четкий "приказ" интерпретатору изменить тип данных. Интерпретатору остается только выполнить это действие.

**Примеры кода:**

*   **Неявное приведение (`string` -> `number`):**
    ```javascript
    '5' - '5' // -> 0 (интерпретатор автоматически преобразует строки в числа для операции вычитания)
    ```
    (Если бы была операция сложения `'5' + '5'`, результат был бы `'55'` из-за конкатенации строк, а не сложения чисел).

*   **Явное приведение (`string` -> `number`):**
    ```javascript
    5 + Number('5') // -> 10 (мы явно указали преобразовать '5' в число)
    ```

#### 2. Явное приведение `number`, `boolean` --> `string`

**Объяснение концепции:**
Для преобразования чисел (number) или булевых значений (boolean) в строки (string) существует несколько явных методов.

**Примеры кода:**

```javascript
let n = -5.5; // Числовая переменная
// 1. Метод .toString()
n.toString(); // -> "-5.5"
// 2. Глобальная функция String()
String(n); // -> "-5.5"
// 3. Конкатенация с пустой строкой
'' + n; // -> "-5.5"

let b = true; // Булевая переменная
// 4. Шаблонный литерал
`${b}`; // -> "true"
```

**Важные замечания:**
Лучше использовать варианты с `.toString()` или `String(...)`, так как они явно описывают выполняемое действие и делают код более читабельным. Конкатенация с пустой строкой или шаблонные литералы работают, но менее очевидны как методы приведения типов.

#### 3. Явное приведение `string` --> `number`

**Объяснение концепции:**
Для преобразования строк (string) в числа (number) также есть несколько методов.

**Примеры кода:**

```javascript
let s = "-5.5"; // Строковая переменная
// 1. Унарный оператор плюс
+s; // -> -5.5 (преобразует строку в число, сохраняя знак)
// 2. Унарный оператор минус (преобразует в число и инвертирует знак)
-s; // -> 5.5
// 3. Глобальная функция Number()
Number(s); // -> -5.5
// 4. Глобальная функция parseFloat()
parseFloat(s); // -> -5.5 (парсит строку и возвращает число с плавающей точкой)
// 5. Глобальная функция parseInt()
parseInt(s); // -> -5 (парсит строку и возвращает целое число)
```

**Важные замечания:**
`parseInt()` и `parseFloat()` нужно применять с осторожностью, изучив их поведение, так как они парсят строку до первого нечислового символа. Например, `parseInt('5s')` вернет `5`, а `parseInt('s5')` вернет `NaN`.
В целом, для общего приведения строки к числу часто предпочитают `Number(s)` или унарный плюс `+s` из-за их более предсказуемого и строгого поведения.

#### 4. Явное приведение `string` --> `boolean`

**Объяснение концепции:**
Для преобразования строк в булевы значения (boolean) используются операторы логического отрицания или глобальная функция `Boolean()`.

**Примеры кода:**

```javascript
let sFalse = ''; // Пустая строка
// 1. Двойное логическое отрицание (double negation)
!!sFalse; // -> false
// 2. Глобальная функция Boolean()
Boolean(sFalse); // -> false

let sTrue = 's'; // Непустая строка
!!sTrue; // -> true
Boolean(sTrue); // -> true
```

**Важные замечания:**
**Надо запомнить (только для JS!!!):**
*   Только пустая строка (`''`) при приведении в булевый тип дает `false`.
*   Любая непустая строка (даже строка с пробелом `' '`) дает `true`.

#### 5. Явное приведение `number` --> `boolean`

**Объяснение концепции:**
Приведение чисел к булевому типу аналогично строкам, но с другим ключевым "ложным" значением.

**Примеры кода:**

```javascript
let nFalse = 0; // Число ноль
!!nFalse; // -> false
Boolean(nFalse); // -> false

let nTrue = 1; // Любое другое число (положительное)
!!nTrue; // -> true
Boolean(nTrue); // -> true

let nNegative = -5; // Отрицательное число
!!nNegative; // -> true
Boolean(nNegative); // -> true

let nDecimal = 0.0001; // Десятичное число, близкое к нулю
!!nDecimal; // -> true
Boolean(nDecimal); // -> true
```

**Важные замечания:**
**Надо запомнить (только для JS!!!):**
*   Только число `0` при приведении в булевый тип дает `false`.
*   Все остальные числа (положительные, отрицательные, десятичные) дают `true`.

#### 6. Явное приведение `object` --> `boolean`

**Объяснение концепции:**
При приведении объектов к булевому типу практически все объекты считаются "истинными" (truthy).

**Примеры кода:**

```javascript
let obj = {}; // Пустой объект
Boolean(obj); // -> true

let arr = []; // Пустой массив
Boolean(arr); // -> true

let newMap = new Map(); // Новый объект Map
Boolean(newMap); // -> true

let x = null; // Значение null
Boolean(x); // -> false
```

**Важные замечания:**
**Надо запомнить (только для JS!!!):**
*   Только объект `null` при приведении в булевый тип дает `false`.
*   Все остальные объекты (включая пустые объекты `{}`, пустые массивы `[]`, новые экземпляры классов/структур) дают `true`.

#### 7. Явное приведение `undefined`, `null` --> `boolean`

**Объяснение концепции:**
Значения `undefined` и `null` являются фундаментальными "ложными" значениями в JavaScript.

**Примеры кода:**

```javascript
let u; // Переменная без присвоенного значения (undefined)
!!u; // -> false
Boolean(u); // -> false

let n = null; // Значение null
!!n; // -> false
Boolean(n); // -> false
```

**Важные замечания:**
*   Значение `undefined` является в булевом эквиваленте ЛОЖНЫМ.
*   Значение `null` является в булевом эквиваленте ЛОЖНЫМ.

### Ключевые выводы

1.  **Приведение типов** — это изменение типа данных во время выполнения кода.
2.  **Явное приведение** — мы явно указываем интерпретатору изменить тип данных (например, `Number()`, `String()`, `Boolean()`, `+variable`, `!!variable`).
3.  **Неявное приведение** — происходит автоматически по правилам JS (будет рассмотрено в следующем видео).
4.  **"Ложные" (falsy) значения:** В JavaScript есть всего несколько значений, которые при приведении к булевому типу дают `false`: `false`, пустая строка (`''`), число `0`, `undefined` и `null`. Все остальные значения (включая непустые строки, любые числа кроме 0, все объекты) дают `true`.

---

### Термины

*   **Приведение типов (Type Coercion):** Процесс изменения типа данных в ходе выполнения программы.
*   **Явное приведение типов (Explicit Type Coercion):** Изменение типа данных, явно указанное разработчиком в коде.
*   **Неявное приведение типов (Implicit Type Coercion):** Автоматическое изменение типа данных, выполняемое интерпретатором JavaScript.
*   **Truthy значения:** Значения, которые при приведении к булевому типу дают `true`.
*   **Falsy значения:** Значения, которые при приведении к булевому типу дают `false`. В JS это: `false`, `0`, `''`, `null`, `undefined`.
*   **Унарный оператор плюс (`+`):** Оператор, используемый для явного преобразования значения к числовому типу.
*   **Унарный оператор минус (`-`):** Оператор, используемый для явного преобразования значения к числовому типу с инвертированием знака.
*   **Двойное логическое отрицание (`!!`):** Комбинация операторов, используемая для явного преобразования значения к булевому типу.
*   **`Number()`:** Глобальная функция для явного преобразования значения к числовому типу.
*   **`String()`:** Глобальная функция для явного преобразования значения к строковому типу.
*   **`Boolean()`:** Глобальная функция для явного преобразования значения к булевому типу.
*   **`parseInt()`:** Глобальная функция для парсинга строки и возвращения целого числа.
*   **`parseFloat()`:** Глобальная функция для парсинга строки и возвращения числа с плавающей точкой.
