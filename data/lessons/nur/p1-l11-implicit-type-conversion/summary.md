# Неявное преобразование типов — краткая выжимка

⏱ **Видео:** 08:17

- **Неявное приведение (coercion)** — автоматическое преобразование типов интерпретатором JS без явного указания
- **В математике:** `+` со строкой → конкатенация; `-`, `*`, `/`, `%` → преобразование в число
- **В условиях (`if`):** значение преобразуется в boolean; **falsy:** `false`, `0`, `''`, `null`, `undefined`, `NaN`
- **Нестрогое равенство `==`** выполняет неявные преобразования — **избегайте, используйте `===`**
- **Понимание coercion критично** для предсказуемого поведения кода и избежания багов
