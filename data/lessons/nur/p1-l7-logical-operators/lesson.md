### Подробный конспект

#### Логические операторы И и ИЛИ в жизни

**Концепция:** В повседневной речи мы часто используем союзы "И" и "ИЛИ" для объединения условий. В зависимости от выбранного союза, результат (решение или действие) будет зависеть от истинности или ложности отдельных частей условия.

**Пример ИЛИ (0:20):**
"Если (сегодня выходной **ИЛИ** праздничный день), то мне не надо идти на работу"
*   **Нюанс:** Чтобы мне не надо было идти на работу, достаточно, чтобы **одно из условий** было истинным (либо выходной, либо праздничный день).

**Пример И (0:56):**
"Если (сегодня рабочий день **И** время между 9 и 18 часами), то я должен быть на работе"
*   **Нюанс:** Чтобы я должен был быть на работе, необходимо, чтобы **оба условия** были истинными (и рабочий день, и время между 9 и 18 часами). Если хотя бы одно из этих условий ложно, то мне не обязательно быть на работе.

#### Логический оператор ИЛИ (||) в JavaScript

**Концепция (1:53):**
Оператор `||` (ИЛИ) объединяет два выражения. Он возвращает `true` (истину), если хотя бы одно из выражений является истинным. В противном случае (если оба выражения ложны), он возвращает `false` (ложь).

**Примеры (Таблица истинности) (2:25):**
```javascript
true || true;   //--> true
true || false;  //--> true
false || true;  //--> true
false || false; //--> false
```

**Пошаговое выполнение кода с ИЛИ (i = 5) (3:17):**
Рассмотрим, как интерпретатор выполняет этот код.
```javascript
let i = 5;
i === 0 || i === 5;
```
1.  Вычисляется первое выражение: `i === 0`. Подставляем значение `i`: `5 === 0`. Это вернет `false`.
2.  Вычисляется второе выражение: `i === 5`. Подставляем значение `i`: `5 === 5`. Это вернет `true`.
3.  Комбинируем результаты: `false || true`. Согласно таблице истинности, это вернет `true`.

**Пошаговое выполнение кода с ИЛИ (i = 3) (4:21):**
```javascript
let i = 3;
i === 0 || i === 5;
```
1.  Вычисляется первое выражение: `i === 0`. Подставляем значение `i`: `3 === 0`. Это вернет `false`.
2.  Вычисляется второе выражение: `i === 5`. Подставляем значение `i`: `3 === 5`. Это вернет `false`.
3.  Комбинируем результаты: `false || false`. Согласно таблице истинности, это вернет `false`.

#### Логический оператор И (&&) в JavaScript

**Концепция (4:41):**
Оператор `&&` (И) объединяет два выражения. Он возвращает `true` (истину), только если **оба** выражения являются истинными. В противном случае (если хотя бы одно ложно), он возвращает `false` (ложь).

**Примеры (Таблица истинности) (5:08):**
```javascript
true && true;   //--> true
true && false;  //--> false
false && true;  //--> false
false && false; //--> false
```

**Пошаговое выполнение кода с И (i = 3) (5:26):**
```javascript
let i = 3;
i >= 0 && i <= 5;
```
1.  Вычисляется первое выражение: `i >= 0`. Подставляем значение `i`: `3 >= 0`. Это вернет `true`.
2.  Вычисляется второе выражение: `i <= 5`. Подставляем значение `i`: `3 <= 5`. Это вернет `true`.
3.  Комбинируем результаты: `true && true`. Согласно таблице истинности, это вернет `true`.

**Пошаговое выполнение кода с И (i = 7) (5:56):**
```javascript
let i = 7;
i >= 0 && i <= 5;
```
1.  Вычисляется первое выражение: `i >= 0`. Подставляем значение `i`: `7 >= 0`. Это вернет `true`.
2.  Вычисляется второе выражение: `i <= 5`. Подставляем значение `i`: `7 <= 5`. Это вернет `false`.
3.  Комбинируем результаты: `true && false`. Согласно таблице истинности, это вернет `false`.

#### Составление сложных логических выражений

**Концепция (6:17):**
Можно составлять более сложные логические выражения, комбинируя операторы `&&` и `||`. Использование скобок `()` помогает явно задать порядок выполнения операций и улучшить читаемость кода. Хотя интерпретатор может выполнить выражение без скобок так же, как и со скобками (в соответствии с приоритетом операторов), для человека чтение кода со скобками часто более понятно.
*   **Нюанс:** Одна из основных задач программиста — писать не только верный, но и понятный код, особенно для других разработчиков, которые будут его читать или изменять. Скобки – один из приемов для достижения этой цели.

**Примеры сложных выражений (6:43):**
```javascript
(true && false) || (false && false)
// Это выражение эквивалентно следующему для интерпретатора:
true && false || false && false
```
*   **Нюанс:** Подробности о приоритете выполнения операторов будут рассмотрены в отдельном уроке.

**Практический пример (Проверка трудоспособного возраста) (9:10):**
Мы хотим определить, находится ли человек в трудоспособном возрасте. Согласно примеру, `isWorkingAgePerson` будет равна `true`, если возраст больше или равен 18 И (меньше 65 для мужчин ИЛИ меньше 63 для женщин).

Код для определения трудоспособного возраста:
```javascript
// Представим, что у нас есть две переменные:
// let personOld;  // возраст человека
// let isMan;      // true, если мужчина; false, если женщина

let isWorkingAgePerson = (personOld >= 18) &&
                         ((isMan === true && personOld < 65) ||
                          (isMan === false && personOld < 63));
```
*   `personOld >= 18`: Возраст должен быть не меньше 18.
*   `isMan === true && personOld < 65`: Если человек мужчина, его возраст должен быть меньше 65.
*   `isMan === false && personOld < 63`: Если человек женщина, его возраст должен быть меньше 63.

**Практические эксперименты в консоли (11:47):**

1.  **Сценарий 1: Мужчина, 15 лет (не трудоспособен)**
    ```javascript
    let personOld = 15;
    let isMan = true;
    let isWorkingAgePerson = (personOld >= 18) &&
                             ((isMan === true && personOld < 65) ||
                              (isMan === false && personOld < 63));
    isWorkingAgePerson; // Вывод: false
    ```
    *   **Объяснение:** Первая часть условия `(personOld >= 18)` становится `(15 >= 18)` которое равно `false`. Поскольку `false && ...` всегда `false`, все выражение возвращает `false`.

2.  **Сценарий 2: Мужчина, 18 лет (трудоспособен)**
    ```javascript
    personOld = 18; // Изменяем возраст
    isMan = true;   // Остается мужчиной

    isWorkingAgePerson = (personOld >= 18) &&
                         ((isMan === true && personOld < 65) ||
                          (isMan === false && personOld < 63));
    isWorkingAgePerson; // Вывод: true
    ```
    *   **Объяснение:**
        *   `(personOld >= 18)` становится `(18 >= 18)`, это `true`.
        *   `((isMan === true && personOld < 65) || (isMan === false && personOld < 63))`
            *   Первая часть `(isMan === true && personOld < 65)` становится `(true === true && 18 < 65)`, что равно `(true && true)`, то есть `true`.
            *   Вторая часть `(isMan === false && personOld < 63)` становится `(true === false && 18 < 63)`, что равно `(false && true)`, то есть `false`.
            *   Итого внутри скобок `(true || false)` становится `true`.
        *   Вся строка: `true && true` возвращает `true`.

### Термины

*   **Логический оператор И (`&&`):** Объединяет два булевых выражения. Возвращает `true`, если *оба* выражения истинны, иначе `false`.
*   **Логический оператор ИЛИ (`||`):** Объединяет два булевых выражения. Возвращает `true`, если *хотя бы одно* из выражений истинно, иначе `false`.
*   **Выражение:** Часть кода, которая может быть вычислена и возвращает какое-либо значение (например, `5 > 3` или `i === 0`).
*   **`true` (истина):** Булево значение, обозначающее правдивость утверждения.
*   **`false` (ложь):** Булево значение, обозначающее неправдивость утверждения.
*   **Булево значение (Boolean):** Тип данных, который может принимать только одно из двух значений: `true` или `false`.
