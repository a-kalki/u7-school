# Операторы in, for…in, for…of — краткая выжимка

⏱ **Чтение:** ~15 мин

- **`in`** — проверяет существование свойства в объекте: `"key" in obj`; надёжнее чем `=== undefined`
- **`for...in`** — перебирает все перечисляемые свойства объекта (включая унаследованные); только для объектов
- **`for...of`** — перебирает значения итерируемых объектов (массивы, строки, Map, Set); рекомендуется для массивов
- **Отличие:** `for...in` — ключи, `for...of` — значения; для массивов всегда `for...of`
- **Правило:** объекты → `for...in`, массивы/строки/Map/Set → `for...of`
