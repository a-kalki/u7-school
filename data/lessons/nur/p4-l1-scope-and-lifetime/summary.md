# Область видимости и время жизни — краткая выжимка

- **Scope** — где переменная доступна. Вне своей области → `ReferenceError`
- **Время жизни** — от создания до уничтожения при выходе из области видимости
- **Глобальная** — вне всего, живёт всю программу. Минимум глобальных переменных
- **Функциональная** — внутри функции (включая параметры). `var` ограничен функцией, а не блоком
- **Блочная** (`let`, `const`) — внутри `{ }`. Каждая итерация `for` создаёт новую переменную
- **Сборщик мусора (GC)** — автоматически удаляет недоступные переменные. Объект жив, пока есть ссылка
- `var` игнорирует блоки и «утекает» из циклов — **не используй**
- Scope — защита от конфликтов и утечек данных
