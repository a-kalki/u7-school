# Функции: продвинутые возможности — краткая выжимка

⏱ **Видео:** 15:14

- **Вложенные функции** — функции внутри функций; доступны только в родительской области видимости
- **Замыкания (closures)** — вложенная функция «запоминает» переменные внешней области даже после её завершения
- **Колбэки** — функции, передаваемые как аргументы; основа асинхронного программирования
- **Функция, возвращающая функцию** — для создания замыканий и ленивых вычислений
- **Ленивые вычисления** — отсрочка выполнения до момента, когда результат действительно нужен
