# Функции: тип, свойства и стили объявления

### Часть 1: Основы функций — тип, свойства и стили объявления

В этой части рассматривается техническая природа функций в JavaScript и различные способы их объявления.

**Ключевые идеи:**
*   **Функции как тип данных:** В JavaScript функция — это особый тип данных (`'function'`), который по своей природе похож на объект.
*   **Свойства функций:** Функции могут иметь собственные свойства, которые можно определять и изменять (например, `callCount`). При вызове функции фактически вызывается её внутренний метод `call()`.
*   **First-class citizens:** Функции могут быть присвоены переменным, переданы как аргументы и возвращены из других функций.
*   **Способы объявления функций:**
    *   **Именованная функция (`function declaration`):** Объявляется с ключевым словом `function` и имеет имя. Поддерживает хойстинг.
    *   **Анонимная функция (`function expression`):** Функцию присваивают переменной. Не имеет имени, не хойстится.
    *   **Стрелочная функция (`arrow function`, ES6+):** Краткий синтаксис, особенности `this`. Если тело состоит из одного выражения, `return` и фигурные скобки можно опустить.

**Примеры кода:**

```javascript
// Функция как объект со свойствами
function sayHi() {
    sayHi.callCount = (sayHi.callCount || 0) + 1;
    return 'Hi';
}
console.log(typeof sayHi);    // -> 'function'
sayHi();
console.log(sayHi.callCount); // -> 1
sayHi.call();
console.log(sayHi.callCount); // -> 2

// Именованная функция
function sayHi1() { return 'Hi 1'; }

// Анонимная функция (присвоенная переменной)
let sayHi2 = function() { return 'Hi 2'; };

// Стрелочная функция с телом
let sayHi3 = () => { return 'Hi 3'; };

// Стрелочная функция с выражением (неявный возврат)
let sayHi4 = () => 'Hi 4';

console.log(sayHi1(), sayHi2(), sayHi3(), sayHi4());
// -> 'Hi 1' 'Hi 2' 'Hi 3' 'Hi 4'
```
