# Циклы: while и do-while

### Часть 3 (23:59-конец): `while`, `do-while` и итоги

**Оператор `while`:**
*   **Синтаксис:**
    ```javascript
    while (cond) {
        cycleBody;
    }
    ```
*   **Особенности:**
    *   Условие (`cond`) проверяется **перед каждой итерацией**.
    *   Если условие `cond` изначально ложно, тело цикла `cycleBody` **не выполнится ни разу**.
    *   Необходимо вручную инициализировать переменные, влияющие на условие (до цикла), и изменять их внутри цикла, чтобы избежать бесконечного цикла.

*   **Пример (`while` для перебора строки):**
    ```javascript
    let h = 'programmer';
    let i = 0; // Инициализация счетчика
    while (i < h.length) { // Условие
        const s = i + ': ' + h[i];
        console.log(s);
        i += 1; // Инкремент счетчика внутри тела цикла
    }
    // Вывод будет таким же, как для for-цикла со строкой.
    ```

**Оператор `do...while`:**
*   **Синтаксис:**
    ```javascript
    do {
        cycleBody;
    } while (cond); // Условие проверяется в конце
    ```
*   **Особенности:**
    *   Тело цикла `cycleBody` **гарантированно выполнится хотя бы один раз**, независимо от начального состояния условия.
    *   Условие (`cond`) проверяется **после** выполнения тела цикла.

*   **Пример (условие сразу ложно):**
    ```javascript
    let h = 'hello world';
    let i = 0;
    do {
        console.log(i + ': ' + h); // Выполнится 1 раз (i=0)
        i += 1; // i станет 1
    } while (i < 0); // Условие (1 < 0) ложно, цикл завершается.
    // Вывод:
    // 0: hello world
    ```

**Ключевые выводы (Итоги):**
*   **`for`:** Выбирается, когда количество итераций известно.
*   **`while`:** Подходит, когда количество итераций заранее неизвестно, и цикл продолжается до выполнения определенного условия. Условие проверяется перед выполнением тела.
*   **`do...while`:** Используется, когда тело цикла должно выполниться **хотя бы один раз**, так как условие проверяется после первой итерации.
*   Выбор типа цикла зависит от требований к инициализации, проверке условия и гарантированному выполнению.

**Основные термины:**
*   **Цикл:** Структура для многократного выполнения кода.
*   **Итерация:** Однократное выполнение тела цикла.
*   **`for`:** Цикл с явной инициализацией, условием и обновлением счетчика.
*   **`while`:** Цикл, повторяющий блок кода, пока условие истинно (проверяется в начале).
*   **`do...while`:** Цикл, выполняющий блок кода один раз, а затем повторяющий его, пока условие истинно (проверяется в конце).
