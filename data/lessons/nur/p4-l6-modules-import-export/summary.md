# Модули: import и export — краткая выжимка

⏱ **Чтение:** ~20 мин

- **Модуль** — отдельный файл со своей областью видимости; выполняется в строгом режиме
- **Named export** — `export function/const`, импорт через `import { name }`
- **Default export** — один на модуль: `export default class/function`, импорт без `{}`
- **Реэкспорт** — `export { ... } from './file.js'` для сборных модулей (barrel)
- **import \* as obj** — импорт всех named-экспортов как объекта
- **Модули выполняются один раз** — результат экспорта кешируется
- **`<script type="module">`** — подключает модуль в браузере (асинхронно)
- **Динамический импорт** — `import('./module.js')` возвращает промис
