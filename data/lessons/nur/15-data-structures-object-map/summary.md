# Структура данных: Object (объект) — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Object** — коллекция пар «ключ: значение», ключ — строка или символ; доступ через `.` или `[]`
- **Проверка свойств:** оператор `in`; перебор — `for...in`
- **Вычисляемые свойства:** `[выражение]: значение`
- **Object.keys/values/entries** — получить ключи, значения или пары из объекта
