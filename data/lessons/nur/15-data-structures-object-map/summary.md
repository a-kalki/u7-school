# Структура данных: ключ-значение (Object, Map) — краткая выжимка

⏱ **Чтение:** ~15 мин

- **Object** — коллекция пар «ключ: значение», ключ — строка или символ; доступ через `.` или `[]`
- **Проверка свойств:** оператор `in`; перебор — `for...in`; вычисляемые свойства `[выражение]`
- **Map** — ключом может быть любое значение (объект, число, boolean); методы: `set`, `get`, `has`, `delete`, `size`
- **Главное отличие:** Map сохраняет тип ключа и порядок вставки; Object приводит ключи к строкам
- **Когда что:** Object — для именованных коллекций; Map — когда нужны нестроковые ключи или частые добавления/удаления
