# Мастер-класс: рисуем пирамиду

### Часть: Функция buildPyramid — пошаговое построение с ментором

Ментор вживую показывает полный цикл разработки функции `buildPyramid(n)`. Студенты повторяют каждый шаг за ментором.

**Ключевые идеи:**
*   **Постановка задачи:** Функция принимает высоту пирамиды `n` (целое положительное число) и возвращает строку, где каждая строка разделена `\n`. Каждый ряд `i` (от 1 до `n`) состоит из `2*i - 1` символов `#`, выровненных по центру пробелами.
*   **Циклы для построения:** Внешний цикл по рядам (`for i = 1..n`), внутренняя логика для пробелов и решёток.
*   **Работа со строками:** Конкатенация символов в строку, накопление результата.
*   **TDD-подход:** Сначала пишем тест, потом реализацию.

**Пример вызова и ожидаемый результат:**

```javascript
console.log(buildPyramid(3));
//   #
//  ###
// #####

console.log(buildPyramid(5));
//     #
//    ###
//   #####
//  #######
// #########
```
