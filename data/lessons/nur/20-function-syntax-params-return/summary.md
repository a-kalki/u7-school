# Функция: синтаксис, параметры и return — краткая выжимка

⏱ **Видео:** 22:00

- **Функция** — именованный блок кода, выполняющий одно действие; объявляется через `function`
- **Параметры** делают функцию гибкой; **аргументы** передаются при вызове
- **`return`** возвращает значение из функции и завершает её выполнение
- **Предикаты** — функции, возвращающие `true`/`false`; именуются с `is`, `has`, `can`
- **Параметры по умолчанию:** `function fn(param = defaultValue)` — если аргумент не передан
