# Математические операторы

**Краткое содержание:**
Этот урок посвящен основным математическим операторам в JavaScript, включая возведение в степень и взятие остатка от деления. Также рассматривается сокращенный синтаксис для комбинированных операций присваивания и особенности префиксных и постфиксных операторов инкремента/декремента, с советами по их использованию.

**Основные темы:**
1.  Математические операторы
2.  Порядок выполнения в строке присваивания
3.  Сокращенный синтаксис математических операторов с присваиванием
4.  Операторы пре(пост)фиксного инкремента и декремента

### Подробный конспект

#### 1. Математические операторы

Математические операторы в JavaScript аналогичны тем, что используются в математике со школьного курса. Они позволяют выполнять арифметические действия над числовыми данными.

*   **Стандартные операторы:**
    *   `+` (сложение)
    *   `-` (вычитание)
    *   `*` (умножение)
    *   `/` (деление)

*   **Примеры использования стандартных операторов:**
    ```javascript
    let bar = 3; // Создаем переменную bar и присваиваем ей значение 3

    // Выполнение операций:
    bar + 1; // Компьютер возьмет значение bar (3), прибавит 1, получит 4 и вернет 4.
             // Важно: значение переменной bar при этом не изменится.
             // bar все еще равно 3.

    bar / 2; // Компьютер возьмет значение bar (3), разделит на 2, получит 1.5 и вернет 1.5.
             // Важно: значение переменной bar при этом не изменится.
             // bar все еще равно 3.
    ```
    *   Процесс: вместо имени переменной `bar` подставляется её текущее значение (3), затем выполняется математическая операция, и результат возвращается.

*   **Дополнительные математические операторы:**
    *   `**` (возведение в степень)
    *   `%` (оператор получения остатка от деления)

*   **Примеры использования дополнительных операторов:**
    ```javascript
    // Допустим, bar сейчас равно 2 (его можно переприсвоить: bar = 2;)
    bar ** 3; // Компьютер возьмет значение bar (2), возведет в 3-ю степень (2*2*2=8) и вернет 8.
              // bar все еще равно 2.

    // Допустим, bar сейчас равно 8 (bar = 8;)
    bar % 3; // Компьютер возьмет значение bar (8), разделит на 3, получит остаток 2 и вернет 2.
             // bar все еще равно 8.
             // (8 разделить на 3 = 2 целых и 2 в остатке)

    // Допустим, bar сейчас равно 5 (bar = 5;)
    bar % 4; // Компьютер возьмет значение bar (5), разделит на 4, получит остаток 1 и вернет 1.
             // bar все еще равно 5.
    ```

#### 2. Порядок выполнения в строке присваивания

Когда компьютер встречает оператор присваивания (`=`), он всегда сначала выполняет правую часть выражения, а затем записывает полученное значение в переменную, указанную в левой части.

*   **Пример:**
    ```javascript
    let y = 7;
    let x;

    x = y + 1; // 1. Вычисляется правая часть: y + 1 -> 7 + 1 -> 8
               // 2. Полученное значение (8) присваивается переменной x.
               // Теперь x равно 8.
    ```
    *   Иными словами, строка `x = y + 1;` выполняется в два шага: сначала вычисление справа, затем присваивание слева.

#### 3. Сокращенный синтаксис математических операторов с присваиванием

Для операций, где результат вычисления присваивается той же переменной, которая участвовала в вычислении, существует сокращенный синтаксис.

*   **Концепция:** `x = x + 1;` можно сократить до `x += 1;`
*   **Механизм:** Сначала считывается текущее значение переменной (`x`), производится вычисление (например, `x + 1`), а затем результат записывается обратно в ту же переменную (`x`).

*   **Примеры сокращенного синтаксиса (работает со всеми математическими операторами):**
    *   `x += 1;` (эквивалентно `x = x + 1;`)
    *   `x -= 3;` (эквивалентно `x = x - 3;`)
    *   `x *= 2;` (эквивалентно `x = x * 2;`)
    *   `x /= 5;` (эквивалентно `x = x / 5;`)
    *   `x %= 3;` (эквивалентно `x = x % 3;`)
    *   `x **= 2;` (эквивалентно `x = x ** 2;`)

*   **Live Demo (в консоли браузера):**
    ```javascript
    let baz = 5;
    baz **= 3; // baz = baz ** 3; -> baz = 5 ** 3; -> baz = 125;
    baz;       // Вывод: 125

    baz %= 3;  // baz = baz % 3; -> baz = 125 % 3; -> baz = 2;
    baz;       // Вывод: 2
    ```

#### 4. Операторы пре(пост)фиксного инкремента и декремента

Эти операторы позволяют увеличить (`++`) или уменьшить (`--`) значение переменной на 1. Различие между префиксной и постфиксной формой заключается в порядке выполнения операций.

*   **Постфиксный оператор инкремента и декремента (`baz++`, `baz--`):**
    1.  Возвращает **текущее** значение переменной.
    2.  После этого увеличивает или уменьшает значение переменной на один.

    *   **Пример (Live Demo):**
        ```javascript
        let baz = 2;
        alert(baz++); // 1. alert() выводит текущее значение baz (2).
                      // 2. Затем baz увеличивается на 1.
                      // Вывод в окне alert: 2
        baz;          // Вывод: 3 (значение baz изменилось)
        ```

*   **Префиксный оператор инкремента и декремента (`++baz`, `--baz`):**
    1.  Увеличивает или уменьшает значение переменной на один.
    2.  После этого возвращает **новое** значение переменной.

    *   **Пример (Live Demo):**
        ```javascript
        let baz = 2;
        alert(++baz); // 1. Сначала baz увеличивается на 1 (становится 3).
                      // 2. Затем alert() выводит новое значение baz (3).
                      // Вывод в окне alert: 3
        baz;          // Вывод: 3 (значение baz изменилось)
        ```

*   **Важное замечание от преподавателя:**
    Лучше **не использовать** операторы `++` и `--` в сложных выражениях или в сочетании с другими операциями. Причина в том, что эти операторы неявно делают два действия (возвращают значение и изменяют переменную), что может приводить к сложнонаходимым ошибкам в коде и ухудшает читаемость.

    *   **Пример сложного кода с инкрементом/декрементом:**
        ```javascript
        diff = arrCount * (col - i++) / --row;
        ```
        Эту строку сложно понять, так как порядок выполнения операций и изменение переменных не сразу очевидны.

    *   **Рекомендуемый способ (более читаемый и предсказуемый):**
        Разбивать сложные выражения на несколько простых строк.
        ```javascript
        row -= 1; // или --row; - сначала уменьшаем row
        diff = arrCount * (col - i) / row; // Затем выполняем основное вычисление, используя уже измененное значение row
        i += 1; // или i++; - После этого увеличиваем i
        ```
        Такой подход делает код гораздо более прозрачным и предотвращает потенциальные ошибки, связанные с порядком операций.

### Ключевые выводы

1.  **Математические операторы:** JavaScript поддерживает стандартные математические операторы (`+`, `-`, `*`, `/`) а также операторы возведения в степень (`**`) и получения остатка от деления (`%`).
2.  **Порядок присваивания:** В строке присваивания (`variable = expression;`) всегда сначала вычисляется правая часть, а затем результат присваивается переменной слева.
3.  **Сокращенный синтаксис:** Используйте сокращенные операторы (`+=`, `-=`, `*=`, `/=`, `%=`, `**=`) для более компактной записи, когда переменная участвует в операции и ей же присваивается результат.
4.  **Инкремент/Декремент:** Постфиксные операторы (`var++`, `var--`) возвращают значение переменной *до* изменения, а префиксные (`++var`, `--var`) возвращают значение *после* изменения.
5.  **Читаемость кода:** Избегайте использования префиксных и постфиксных операторов инкремента/декремента в сложных выражениях, чтобы код был более читаемым и легким для отладки.

### Термины

*   **Оператор `**` (Возведение в степень):** Вычисляет основание, возведенное в степень показателя. Например, `2 ** 3` равно 8.
*   **Оператор `%` (По модулю / Остаток от деления):** Возвращает остаток, когда одно число делится на другое. Например, `8 % 3` равно 2.
*   **Оператор присваивания (`=`):** Присваивает значение правой части переменной, указанной в левой части.
*   **Сокращенные операторы присваивания (`+=`, `-=`, `*=`, `/=`, `%=`, `**=`):** Комбинируют арифметическую операцию с присваиванием. Например, `x += 1` эквивалентно `x = x + 1`.
*   **Постфиксный инкремент/декремент (`var++`, `var--`):** Сначала возвращает текущее значение переменной, затем увеличивает/уменьшает её на 1.
*   **Префиксный инкремент/декремент (`++var`, `--var`):** Сначала увеличивает/уменьшает переменную на 1, затем возвращает её новое значение.
*   **Инкремент:** Увеличение значения переменной на единицу.
*   **Декремент:** Уменьшение значения переменной на единицу.
