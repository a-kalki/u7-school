# Функции: продвинутые возможности

### Часть 3: Продвинутые функциональные возможности

В этой части рассматриваются более сложные концепции, такие как вложенные функции, колбэки и функции, возвращающие другие функции.

**Ключевые идеи:**
*   **Функция внутри функции (вложенные функции):** Функции можно объявлять внутри других функций. Вложенная функция доступна только внутри родительской и полезна для инкапсуляции логики.
*   **Замыкания (`Closures`):** Вложенные функции "запоминают" и имеют доступ к переменным из их внешней области видимости, даже после завершения выполнения внешней функции.
*   **Ссылки на функции и колбэки:** Функции можно передавать в другие функции как аргументы (колбэки). Функция, принимающая колбэк, называется "функцией высшего порядка". Колбэки широко используются для асинхронного кода и обработки событий.
*   **Функция, возвращающая функцию:** Функция может возвращать другую функцию в качестве результата. Это используется для создания замыканий, ленивых вычислений или конфигурирования поведения.
*   **Ленивые вычисления (`Lazy evaluation`):** Отсрочка выполнения ресурсоемких вычислений до момента, когда их результат действительно понадобится.

**Примеры кода:**

```javascript
// Функция внутри функции (вложенная)
function multiply(a, b) {
    function add(x, y) { // Вложенная функция
        return x + y;
    }
    let result = 0;
    for (let i = 0; i < b; i++) {
        result = add(result, a);
    }
    return result;
}
console.log(multiply(4, 3)); // -> 12
// add(2, 3); // -> Uncaught TypeError: add is not a function

// Колбэки (передача функции как аргумента)
let sayHello = () => console.log('Hello!');
function executeFn(callback) {
    console.log('Doing something...');
    callback(); // Вызываем переданную функцию
    console.log('Finished something.');
}
executeFn(sayHello);
// -> Doing something...
// -> Hello!
// -> Finished something.

// Функция, возвращающая функцию (с замыканием и ленивыми вычислениями)
function createCounter(initialValue) {
    let count = initialValue; // Замыкание: count запоминается
    return function() { // Возвращаемая функция
        count++;
        console.log('Current count:', count);
    };
}
let counter1 = createCounter(0); // initialValue = 0 запоминается для counter1
let counter2 = createCounter(10); // initialValue = 10 запоминается для counter2

counter1(); // -> Current count: 1
counter1(); // -> Current count: 2
counter2(); // -> Current count: 11 (независимый счетчик)
```
