# Тернарный оператор — краткая выжимка

⏱ **Видео:** 11:40

- **`if` — обязателен;** `else if` и `else` — опциональны, не могут использоваться без `if`
- **В цепочке if-else if-else** выполняется только первый истинный блок, затем выход
- **Последовательные `if`** (без `else`) проверяются независимо — может выполниться несколько блоков
- **Тернарный оператор `? :`** — компактная замена `if-else`: `условие ? значениеЕслиДа : значениеЕслиНет`
- **Домашнее задание:** самостоятельно изучить `switch...case`
