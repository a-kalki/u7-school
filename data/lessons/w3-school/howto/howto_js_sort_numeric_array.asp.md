# How TO - Sort Numeric Array

* * *

Learn how to sort an array numerically in JavaScript.

* * *

## Sort an Array Numerically

You can use the folllowing to sort an array numerically:

```javascript
const points = [40, 100, 1, 5, 25, 10];points.sort(function(a, b){return a - b});
```

You can also sort the array descending:

```javascript
const points = [40, 100, 1, 5, 25, 10];points.sort(function(a, b){return b - a});
```

Read more about how to sort arrays in our [JavaScript Sorting Arrays Tutorial](https://www.w3schools.com/js/js_array_sort.asp).
