## How To Create Notes

##### Step 1) Add HTML:

```javascript
<!-- Add icon library --><link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css"><span class="fa fa-star checked"></span><span class="fa fa-star checked"></span><span class="fa fa-star checked"></span><span class="fa fa-star"></span><span class="fa fa-star"></span>
```

* * *

##### Step 2) Add CSS:

```javascript
.checked {  color: orange;}
```

**Tip:** Go to our [Icons Tutorial](https://www.w3schools.com/icons/default.asp) to learn more about icons.
