* * *

## Disable Resizing of Textarea

You can use the `resize` property to specify whether a textarea should be resizable or not:

```javascript
textarea {  resize: none;}
```

* * *

**Tip:** Go to our [CSS resize Reference](https://www.w3schools.com/cssref/css3_pr_resize.asp) to learn more the resize property.
