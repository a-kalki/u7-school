To create a zebra-striped table, use the `nth-child()` selector and add a background-color to all even (or odd) table rows:

```javascript
tr:nth-child(even) {  background-color: #f2f2f2;}
```

Go to our [CSS Tables Tutorial](https://www.w3schools.com/css/css_table.asp) to learn more about how to style tables.

Go to our [CSS Selector Reference](https://www.w3schools.com/cssref/sel_nth-child.asp) to learn more about the nth-child() selector.
