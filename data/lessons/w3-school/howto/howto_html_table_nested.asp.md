## Nested Table

A table inside another table:

```javascript
<table>  <tr>    <td>Cell</td>    <td>Nested Table      <table>        <tr>          <td>Cell 1</td>        </tr>        <tr>          <td>Cell 2</td>        </tr>        <tr>          <td>Cell 3</td>        </tr>        <tr>          <td>Cell 4</td>        </tr>      </table>    </td>  </tr></table>
```

**Tip:** Go to our [HTML Table Tutorial](https://www.w3schools.com/html/html_tables.asp) to learn more about tables.
