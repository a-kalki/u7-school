# HTML and CSS Quiz

* * *

You can test your HTML & CSS skills with W3Schools' Quiz.

* * *

## The Test

The test contains 25 questions and there is no time limit.

The test is not official, it's just a nice way to see how much you know, or don't know, about HTML & CSS.

## Count Your Score

You will get 1 point for each correct answer.

At the end of the Quiz, your total score will be displayed.

Maximum score is 25 points.

## Start the Quiz

Good luck!

[Start the HTML & CSS Quiz ❯](https://www.w3schools.com/quiztest/quiztest.asp?qtest=IHTMLCSS)

If you don't know HTML & CSS, we suggest that you read our [HTML & CSS Tutorial](default.asp.html) from scratch.
