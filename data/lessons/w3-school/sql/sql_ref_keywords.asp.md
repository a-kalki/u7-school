# SQL Keywords Reference

* * *

This SQL keywords reference contains the reserved words in SQL.

* * *

## SQL Keywords

Keyword

Description

[ADD](sql_ref_add.asp.html)

Adds a column in an existing table

[ADD CONSTRAINT](sql_ref_add_constraint.asp.html)

Adds a constraint after a table is already created

[ALL](sql_ref_all.asp.html)

Returns true if all of the subquery values meet the condition

[ALTER](sql_ref_alter.asp.html)

Adds, deletes, or modifies columns in a table, or changes the data type of a column in a table

[ALTER COLUMN](sql_ref_alter_column.asp.html)

Changes the data type of a column in a table

[ALTER TABLE](sql_ref_alter_table.asp.html)

Adds, deletes, or modifies columns in a table

[ALTER VIEW](sql_ref_alter_view.asp.html)

Updates a view

[AND](sql_ref_and.asp.html)

Only includes rows where both conditions is true

[ANY](sql_ref_any.asp.html)

Returns true if any of the subquery values meet the condition

[AS](sql_ref_as.asp.html)

Renames a column or table with an alias

[ASC](sql_ref_asc.asp.html)

Sorts the result set in ascending order

[BACKUP DATABASE](sql_ref_backup_database.asp.html)

Creates a back up of an existing database

[BETWEEN](sql_ref_between.asp.html)

Selects values within a given range

[CASE](sql_ref_case.asp.html)

Creates different outputs based on conditions

[CHECK](sql_ref_check.asp.html)

A constraint that limits the value that can be placed in a column

[COLUMN](sql_ref_column.asp.html)

Changes the data type of a column or deletes a column in a table

[CONSTRAINT](sql_ref_constraint.asp.html)

Adds or deletes a constraint

[CREATE](sql_ref_create.asp.html)

Creates a database, index, view, table, or procedure

[CREATE DATABASE](sql_ref_create_database.asp.html)

Creates a new SQL database

[CREATE INDEX](sql_ref_create_index.asp.html)

Creates an index on a table (allows duplicate values)

[CREATE OR REPLACE VIEW](sql_ref_create_or_replace_view.asp.html)

Updates a view

[CREATE TABLE](sql_ref_create_table.asp.html)

Creates a new table in the database

[CREATE PROCEDURE](sql_ref_create_procedure.asp.html)

Creates a stored procedure

[CREATE UNIQUE INDEX](sql_ref_create_unique_index.asp.html)

Creates a unique index on a table (no duplicate values)

[CREATE VIEW](sql_ref_create_view.asp.html)

Creates a view based on the result set of a SELECT statement

[DATABASE](sql_ref_database.asp.html)

Creates or deletes an SQL database

[DEFAULT](sql_ref_default.asp.html)

A constraint that provides a default value for a column

[DELETE](sql_ref_delete.asp.html)

Deletes rows from a table

[DESC](sql_ref_desc.asp.html)

Sorts the result set in descending order

[DISTINCT](sql_ref_distinct.asp.html)

Selects only distinct (different) values

[DROP](sql_ref_drop.asp.html)

Deletes a column, constraint, database, index, table, or view

[DROP COLUMN](sql_ref_drop_column.asp.html)

Deletes a column in a table

[DROP CONSTRAINT](sql_ref_drop_constraint.asp.html)

Deletes a UNIQUE, PRIMARY KEY, FOREIGN KEY, or CHECK constraint

[DROP DATABASE](sql_ref_drop_database.asp.html)

Deletes an existing SQL database

[DROP DEFAULT](sql_ref_drop_default.asp.html)

Deletes a DEFAULT constraint

[DROP INDEX](sql_ref_drop_index.asp.html)

Deletes an index in a table

[DROP TABLE](sql_ref_drop_table.asp.html)

Deletes an existing table in the database

[DROP VIEW](sql_ref_drop_view.asp.html)

Deletes a view

[EXEC](sql_ref_exec.asp.html)

Executes a stored procedure

[EXISTS](sql_ref_exists.asp.html)

Tests for the existence of any record in a subquery

[FOREIGN KEY](sql_ref_foreign_key.asp.html)

A constraint that is a key used to link two tables together

[FROM](sql_ref_from.asp.html)

Specifies which table to select or delete data from

[FULL OUTER JOIN](sql_ref_full_outer_join.asp.html)

Returns all rows when there is a match in either left table or right table

[GROUP BY](sql_ref_group_by.asp.html)

Groups the result set (used with aggregate functions: COUNT, MAX, MIN, SUM, AVG)

[HAVING](sql_ref_having.asp.html)

Used instead of WHERE with aggregate functions

[IN](sql_ref_in.asp.html)

Check if a specified column's value matches any value within a provided list

[INDEX](sql_ref_index.asp.html)

Creates or deletes an index in a table

[INNER JOIN](sql_ref_inner_join.asp.html)

Returns rows that have matching values in both tables

[INSERT INTO](sql_ref_insert_into.asp.html)

Inserts new rows in a table

[INSERT INTO SELECT](sql_ref_insert_into_select.asp.html)

Copies data from one table into another table

[IS NULL](sql_ref_is_null.asp.html)

Tests for empty values

[IS NOT NULL](sql_ref_is_not_null.asp.html)

Tests for non-empty values

[JOIN](sql_ref_join.asp.html)

Joins tables

[LEFT JOIN](sql_ref_left_join.asp.html)

Returns all rows from the left table, and the matching rows from the right table

[LIKE](sql_ref_like.asp.html)

Searches for a specified pattern in a column

[LIMIT](sql_ref_limit.asp.html)

Specifies the number of records to return in the result set

[NOT](sql_ref_not.asp.html)

Only includes rows where a condition is not true

[NOT NULL](sql_ref_not_null.asp.html)

A constraint that enforces a column to not accept NULL values

[OR](sql_ref_or.asp.html)

Includes rows where either condition is true

[ORDER BY](sql_ref_order_by.asp.html)

Sorts the result set in ascending or descending order

[OUTER JOIN](sql_ref_outer_join.asp.html)

Returns all rows when there is a match in either left table or right table

[PRIMARY KEY](sql_ref_primary_key.asp.html)

A constraint that uniquely identifies each record in a database table

[PROCEDURE](sql_ref_procedure.asp.html)

A stored procedure

[RIGHT JOIN](sql_ref_right_join.asp.html)

Returns all rows from the right table, and the matching rows from the left table

[ROWNUM](sql_ref_rownum.asp.html)

Specifies the number of records to return in the result set

[SELECT](sql_ref_select.asp.html)

Selects data from a database

[SELECT DISTINCT](sql_ref_select_distinct.asp.html)

Selects only distinct (different) values

[SELECT INTO](sql_ref_select_into.asp.html)

Copies data from one table into a new table

[SELECT TOP](sql_ref_select_top.asp.html)

Specifies the number of records to return in the result set

[SET](sql_ref_set.asp.html)

Specifies which columns and values that should be updated in a table

[TABLE](sql_ref_table.asp.html)

Creates a table, or adds, deletes, or modifies columns in a table, or deletes a table or data inside a table

[TOP](sql_ref_top.asp.html)

Specifies the number of records to return in the result set

[TRUNCATE TABLE](sql_ref_truncate_table.asp.html)

Deletes the data inside a table, but not the table itself

[UNION](sql_ref_union.asp.html)

Combines the result set of two or more SELECT statements (only distinct values)

[UNION ALL](sql_ref_union_all.asp.html)

Combines the result set of two or more SELECT statements (allows duplicate values)

[UNIQUE](sql_ref_unique.asp.html)

A constraint that ensures that all values in a column are unique

[UPDATE](sql_ref_update.asp.html)

Updates existing rows in a table

[VALUES](sql_ref_values.asp.html)

Specifies the values of an INSERT INTO statement

[VIEW](sql_ref_view.asp.html)

Creates, updates, or deletes a view

[WHERE](sql_ref_where.asp.html)

Filters a result set to include only records that fulfill a specified condition
