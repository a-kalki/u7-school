# Спецификация трека: Stream Guest & Student (US-1..US-5)

## Обзор
Доработать сторис «Витрина», «Карточка», «Запись», «Обучение» и «Прогресс» до полного соответствия `user-stories.md`.

**Роли:** GUEST, SUBSCRIBER, CANDIDATE, STUDENT.

## Функциональные требования

### US-1: Фильтр статусов в CatalogStory
- `handleCallback('list')` должен запрашивать только потоки со статусами `enrollment` и `active`.
- Передавать фильтр в `list-streams` UC. Если API не поддерживает фильтр по нескольким статусам — делать два запроса и объединять.

### US-2: Ролевые кнопки в ViewStreamStory
- **Для GUEST/CANDIDATE на `enrollment`:**
  - Кнопки: `[📝 Записаться]`, `[📖 Программа курса]`, `[⬅️ Назад к списку]`
- **Для GUEST/CANDIDATE на `active`:**
  - Кнопки: `[🔔 Уведомить о наборе]`, `[⬅️ Назад к списку]`
- **Для STUDENT/MENTOR:**
  - Кнопка «Записаться» скрывается
- **Имя ментора:** резолвить через `UserFacade.getUserByUuid(stream.mentorId)`.
- **Кнопка «Программа курса»:** новый callback `view-stream:program:<streamId>` — рендерит `contentSnapshot` в сжатом виде (только заголовки проектов/уроков, без шагов). Под программой кнопка `[⬅️ Назад к потоку]`.

### US-3: Дата старта в EnrollStory
- Добавить `startDate` в сообщение об успешной записи: «Обучение начнётся {StartDate}».
- Данные уже есть в ответе `get-stream`, нужно только отформатировать и вывести.

### US-4: Названия уроков/проектов в LearningStory
- При переходе урока: «Урок {OldLessonTitle} завершён. Начинаем: {NewLessonTitle}».
- При переходе проекта: «Проект {OldProjectTitle} завершён. Начинаем: {NewProjectTitle}».
- Нужно получить названия из `contentSnapshot` (текущий + следующий контекст).

### US-5: Полные данные в ProgressStory
- Добавить: имя ментора, дату старта, ссылку на чат группы.
- Добавить: текущий проект и урок.

## Критерии приёмки
- [ ] Гость видит витрину только с `enrollment` и `active` потоками
- [ ] Гость на карточке `enrollment` видит кнопки «Записаться», «Программа», «Назад»
- [ ] Гость на карточке `active` видит «Уведомить», «Назад»
- [ ] Студент/Ментор не видят кнопку «Записаться»
- [ ] Кнопка «Программа курса» показывает структуру проектов/уроков
- [ ] Сообщение о записи содержит дату старта
- [ ] Переходы между уроками содержат названия уроков
- [ ] Прогресс показывает имя ментора, дату, чат, текущий проект/урок

## За рамками
- Навигация ◀️/▶️ по истории шагов (трек 4)
- Double-confirm (трек 4)
- Менторские кнопки (трек 3)
