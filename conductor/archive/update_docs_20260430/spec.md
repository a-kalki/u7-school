# Specification: Обновление документации и переименование проекта

**Обзор**
Актуализация документации проекта и переименование для перехода к бренду "u7-school". Внедрение информации об использовании методологии Conductor.

**Функциональные требования**
1. **README.md**:
   - Изменить заголовок на `# Курсы u7-school`.
   - Удалить все упоминания "w3c" и "w3schools".
   - Указать, что курсы универсальны и базируются на материалах из различных источников.
   - Добавить раздел об управлении проектом через [Conductor](https://github.com/epm-dev-kalki/conductor).
2. **package.json**:
   - Изменить значение поля `name` на `u7-school`.

**Критерии приемки**
- В файлах `README.md` и `package.json` нет упоминаний "w3c" и "w3schools".
- `README.md` корректно отображает новый бренд и содержит ссылку на документацию Conductor.
- Поле `name` в `package.json` обновлено.

**Вне рамок (Out of Scope)**
- Глобальный поиск и замена во всем исходном коде (изменения только в `README.md` и `package.json`).
- Изменение логики приложения или структуры базы данных.
