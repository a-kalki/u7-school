# Трек: Множественные роли, переименование модуля/файлов, исправление ai-service

## Обзор
Комплексный трек, включающий:
1. Feature: поддержка нескольких ролей у пользователя
2. Refactor: переименование пакета `@u7/core` → `@u7/course`
3. Chore: переход на kebab-case имена файлов в `packages/` и `apps/`
4. Fix: исправление тестов `ai-service` (моки под Gemini → OpenAI/DeepSeek)

## Функциональные требования

### FR1: Множественные роли пользователя
- Поле `role: Role` заменяется на `roles: Role[]`
- Массив должен содержать минимум 1 роль
- `Role` остаётся enum: `STUDENT`, `MENTOR`, `ADMIN`
- Обновить `UserSchema` (Valibot): `v.array(RoleSchema)` с `v.minLength(1)`
- Обновить `UserAr` (агрегат), `UserPolicy`, все use-case'ы и репозитории
- Обновить все существующие тесты

### FR2: Переименование пакета `@u7/core` → `@u7/course`
- Директория: `packages/core` → `packages/course`
- Имя пакета в `package.json`: `@u7/core` → `@u7/course`
- Все импорты `@u7/core` в проекте → `@u7/course`
- Все ссылки в документации и конфигах
- Полная работоспособность после переименования (тесты проходят)

### FR3: kebab-case имена файлов
- Переименовать все `.ts` файлы в `packages/` и `apps/`:
  - `user_ar.ts` → `user-ar.ts`
  - `course_policy.ts` → `course-policy.ts`
  - `iso_now.ts` → `iso-now.ts`
  - `create_user_command.ts` → `create-user-command.ts`
  - ...и т.д.
- Обновить все импорты
- Добавить соглашение в документацию проекта (`docs/architecture.md`)

### FR4: Исправление тестов ai-service
- Текущая проблема: тесты в `ai-service.test.ts` мокают `@google/genai` (Gemini),
  а реальный код использует `openai` (DeepSeek)
- Заменить `mock.module("@google/genai", ...)` на `mock.module("openai", ...)`
- Мок должен эмулировать `chat.completions.create` API OpenAI
- Тест ошибки: использовать актуальную структуру OpenAI (не `ai.models.generateContent`)
- Убрать реальные HTTP-запросы из тестов

## Нефункциональные требования
- Сохранить покрытие тестов >80%
- Все существующие тесты должны проходить после изменений
- **КРИТИЧНО: отсутствие регресса** — ни один существующий тест не должен сломаться
- Обратная совместимость: `@u7/course` экспортирует тот же API
