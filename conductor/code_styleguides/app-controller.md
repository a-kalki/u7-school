# AppController — контроллер уровня приложения

## Назначение

`AppController` — контроллер в `packages/app/src/ui/(bot,rest,cli)`, который обрабатывает пользовательские сценарии (stories), **не привязанные к конкретному доменному модулю** и обрабатывают сценарии для приложения, например предоставить ссылку на группу сообщества школы.
