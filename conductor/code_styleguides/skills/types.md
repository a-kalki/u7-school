# Вспомогательные типы

## Назначение

Допускается создавать дополнительные файлы типов, например `domain/<entity>/types.ts`, которые содержат типы-псевдонимы и вспомогательные определения, которые не вошли в другие файлы предусмотренные фреймворком.

## Когда использовать

- Идентификаторы (`UserId`, `TelegramId`) — семантические обёртки над примитивами.
- Union-типы, специфичные для разных объектов.
- Вспомогательные интерфейсы.

## Пример

```typescript
/** Идентификатор пользователя (UUID) */
export type UserId = string;

/** Telegram ID пользователя */
export type TelegramId = number;
```
