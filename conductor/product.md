# Руководство по продукту

## Целевая аудитория
Менторы: опытные специалисты, которые делятся своими навыками через разработанные курсы в рамках платформы **u7-school**.
Студенты курса: ученики, проходящие обучение, которым требуется доступ к материалам, проверка заданий и прямая связь с менторами.

## Ключевые функции
- **Функционал для создания курсов:** Универсальные инструменты и API для авторов и менторов, позволяющие создавать, редактировать и управлять образовательным контентом, основанным на различных источниках.
- **Умный Telegram-помощник (Grammy):** Бот, сопровождающий студентов в процессе обучения на платформе **u7-school**.
- **REST API (Elysia):** Веб-интерфейс / API для взаимодействия базы данных и других сервисов (например, полноценного веб-приложения) с ядром платформы.
- **Консольный интерфейс (Auto-UI CLI):** Быстрый интерактивный интерфейс для управления системой, автоматически строящийся на основе метаданных модулей.
- **Валидация данных (Valibot):** Строгая проверка данных при создании курсов и ответах пользователей для обеспечения стабильности платформы.

## Стиль общения
Дружелюбный: бот общается в легкой, располагающей манере, поддерживает диалог и использует эмодзи, создавая комфортную и мотивирующую среду для обучения.
