# Руководство по продукту

## Целевая аудитория
- **Гости (GUEST):** Пользователи, только что начавшие взаимодействие с ботом (например, нажали /start). Имеют минимальный доступ и могут стать подписчиками или кандидатами.
- **Подписчики (SUBSCRIBER):** Пользователи, подписанные на Telegram-канал школы. Получают анонсы и могут перейти к подаче заявки.
- **Кандидаты (CANDIDATE):** Пользователи, заполнившие onboarding-анкету и подавшие заявку на обучение. Ожидают зачисления в поток.
- **Студенты курса (STUDENT):** Ученики, проходящие обучение, которым требуется доступ к материалам, проверка заданий и прямая связь с менторами.
- **Менторы:** Опытные специалисты, которые делятся своими навыками через разработанные курсы в рамках платформы **u7-school**.
- **Администраторы (ADMIN):** Управляют пользователями, курсами и заявками.

## Ключевые функции
- **Onboarding и пулл кандидатов:** Система сбора заявок от потенциальных студентов через структурированную анкету (источник, опыт, цели, формат, интенсивность). При подаче заявки пользователю автоматически присваивается роль CANDIDATE. Поддерживает атомарные транзакции.
- **Функционал для создания курсов:** Универсальные инструменты и API для авторов и менторов, позволяющие создавать, редактировать и управлять образовательным контентом, основанным на различных источниках.
- **Умный Telegram-помощник (Grammy):** Бот, сопровождающий студентов в процессе обучения на платформе **u7-school**.
- **REST API (Elysia):** Веб-интерфейс / API для взаимодействия базы данных и других сервисов (например, полноценного веб-приложения) с ядром платформы.
- **Консольный интерфейс (CLI):** Быстрый интерактивный интерфейс для управления системой, автоматически строящийся на основе метаданных модулей.
- **Валидация данных (Valibot):** Строгая проверка данных при создании курсов, ответах пользователей и onboarding-анкетах для обеспечения стабильности платформы.

## Стиль общения
Дружелюбный: бот общается в легкой, располагающей манере, поддерживает диалог и использует эмодзи, создавая комфортную и мотивирующую среду для обучения.
