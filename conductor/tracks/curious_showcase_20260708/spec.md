# Спецификация трека: Витрина для любопытного

> **Контекст эволюции:** `conductor/architecture-evolution.md` (§2.1, §2.2). Прочитать перед началом.
> **Зависимости:** трек 0 (Course), трек 2 (ContentPath — curious-уровень доступа).

## Обзор
Дать «любопытному» (GUEST/CANDIDATE) полную картину для решения о записи: каталог курсов поверх потоков, программа курса (все модули/этапы), расширенная программа модуля (объём, типы шагов), доступность программы/деталей на любом статусе потока, предзапись на completed.

**Роли:** GUEST, CANDIDATE (любопытный); MENTOR/ADMIN тоже видят.

## Функциональные требования

### F1. Каталог курсов (S00) — новый верхний уровень
- Главное меню: «📚 Программы курсов» (над/рядом «📚 Наши потоки»).
- S00: список опубликованных курсов → карточка курса (S00a): title, description, phases с направлениями (tech/business), сводка объёма.
- Программа курса (S00a): phases → modules → projects → lessons (через `get-course`/`resolve-content-path` на curious-уровне).

### F2. Расширенная программа модуля
- Число шагов урока: `📝 Урок (5 шагов)`.
- Типы шагов иконками (текст/код/файл).
- Сводка курса: «N проектов · M уроков · K шагов».
- Контент шагов — скрыт, видны только заголовки (curious-уровень ContentPath).

### F3. Программа/Детали потока на любом статусе
- В S02 «📖 Программа курса» и «📋 Детали» показывать на `enrollment`/`active`/`completed` (сейчас только enrollment).
- На completed — чтобы понять, что проходили выпускники.

### F4. Предзапись на completed-поток
- «🔔 Ждать новый набор» на completed-потоке (модуль которого есть в курсе) → подписка на уведомление о новом потоке этого модуля.
- (Хранилище подписок — мини-репо; уведомление — через TgFacade при появлении enrollment-потока, если решим.)

### F5. «Инструменты ментора» и «Инструменты автора» (входит в этот релиз, но отдельная задача)
- Главное меню MENTOR: «🛠️ Инструменты ментора» → «➕ Создать поток», «📋 Мои потоки».
- Главное меню AUTHOR: «✍️ Инструменты автора» → создание/редактирование модулей, курсов, уроков, шагов.
- Секции role-gated; пользователь с обеими ролями видит обе.
- `CreateStreamStory.handleStart` → второй уровень.

## Критерии приёмки
- [ ] Любопытный видит каталог курсов и программу курса (без контента шагов).
- [ ] Программа модуля показывает объём (шаги, типы).
- [ ] Программа/Детали потока доступны на active/completed.
- [ ] Предзапись работает (или заглушка с уведомлением «сообщим»).
- [ ] «Инструменты ментора» группируют менторские действия.
- [ ] Покрытие >80%, TDD.

## За рамками
- Отзывы выпускников (бэклог).
- Полноценная веб-версия витрины.
- Реальная рассылка уведомлений о наборе (нужен TgFacade + scheduler — бэклог).
