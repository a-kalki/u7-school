# Спецификация трека: Bot UX Polish

## Обзор
Четыре небольших UX-улучшения Telegram-бота u7-school:
1. Кнопка «Сообщество школы» в главном меню.
2. Кнопка «↩️ Назад» на всех уровнях глубже главного меню.
3. Команда `/help` с описанием пунктов главного меню.
4. Легенда цветных кружков в сообщении «Потоки школы».

**Роли:** все пользователи (GUEST, SUBSCRIBER, CANDIDATE, STUDENT, MENTOR, ADMIN).

## Функциональные требования

### FR-1: Кнопка «Сообщество школы»
- В главном меню (после `/start`) добавляется кнопка **«💬 Сообщество школы»**.
- При нажатии — переход по ссылке на Telegram-группу школы (аналог существующей команды `/link_to_school_group`).
- Кнопка доступна всем ролям.
- Приоритет в меню: низкий (после основных пунктов).

### FR-2: Кнопка «↩️ Назад» в иерархии меню
- На любом уровне глубже главного меню (т.е. когда пользователь находится внутри story — каталог, просмотр потока, моя учёба и т.д.) в клавиатуру добавляется кнопка **«↩️ Назад /start»**.
- Нажатие возвращает пользователя в главное меню (эмулирует `/start`).
- На верхнем уровне (само главное меню) кнопка «Назад» не показывается.

### FR-3: Команда /help
- Добавляется обработчик команды `/help` в роутер бота.
- При вызове `/help` бот собирает описания пунктов меню от всех контроллеров:
  - Контроллеры, имеющие `handleHelpStart()`, возвращают строку-описание своих пунктов.
  - Контроллеры без `handleHelpStart()` пропускаются.
- Бот выводит список всех пунктов с описаниями.
- Метод `handleHelpStart()` опционален (по умолчанию возвращает `null`).

### FR-4: Легенда цветных кружков в «Потоки школы»
- В сообщении CatalogStory («📚 Потоки школы») после списка потоков добавляется текст-легенда:
  ```
  🟢 — идёт набор   🔵 — идёт обучение   ⚪ — завершён
  ```

## Критерии приёмки
- [ ] Кнопка «💬 Сообщество школы» отображается в главном меню и работает
- [ ] Кнопка «↩️ Назад /start» появляется на втором уровне меню и возвращает в /start
- [ ] Команда `/help` выводит список пунктов меню с описаниями
- [ ] В «Потоки школы» добавлена легенда цветных кружков
- [ ] Все существующие тесты проходят

## За рамками
- Кастомизация легенды под другие статусы потоков (completed, archived)
- Контекстный help для конкретных story
