# Specification: Перевод проекта в монорепозиторий и интеграция w3school

## Overview
Данный трек направлен на реорганизацию текущего проекта `u7-school` в структуру монорепозитория с использованием Bun Workspaces. Также необходимо интегрировать существующий проект `w3school` из локальной директории в качестве нового пакета.

## Functional Requirements
- Реорганизовать корневую структуру для поддержки Bun Workspaces.
- Создать директории `apps/` и `packages/`
- Перенести текущий код `u7-school` в `apps/u7-bot`
- Перенести проект из `~/Downloads/w3school` в `packages/w3school`
- Настроить общие конфигурации (Biome, TypeScript) для монорепозитория.

## Acceptance Criteria
- [ ] Проект успешно инициализирован как Bun Workspace.
- [ ] Пакет `w3school` перенесен и доступен.
- [ ] `bun run build` проходит успешно.
- [ ] Biome линтинг проходит успешно во всех пакетах.
