# How TO - Get Current URL With JavaScript

* * *

Learn how to get the current URL with JavaScript.

* * *

## Current URL

Use `window.location.href` to get the current URL address:

```javascript
document.getElementById("demo").innerHTML ="The full URL of this page is:<br>" + window.location.href;
```

Read more about the `window.location` object in our [JavaScript Window Location Tutorial](https://www.w3schools.com/js/js_window_location.asp).
