# How TO - Black and White Images

* * *

Learn how to create a "black and white" image with CSS.

* * *

## Black and White Image

Original image

![Pineapple](pineapple.jpg)

grayscale(100%)

![Pineapple](pineapple.jpg)

Use the CSS `filter` property to convert an image to black and white.

```javascript
img {  -webkit-filter: grayscale(100%); /* Safari 6.0 - 9.0 */  filter: grayscale(100%);}
```

Go to our [CSS Images Tutorial](https://www.w3schools.com/css/css3_images.asp) to learn more about how to style images.

Go to our [CSS filter Property](https://www.w3schools.com/cssref/css3_pr_filter.asp) to learn more about CSS filters.
