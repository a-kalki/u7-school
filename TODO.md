## UI основанный на user-story

Это заготовка на возможную будущую имплементацию более человечного ui основанного на user-story.

## Описание
- на стороне фронта вся логика строится вокруг user-story (Us);
- модуль ui может предоставить пользователю список user-story и стандартный трек взаимодействия с модулем;
- user-story имеет всю необходимую логику по сбору данных для выполнения конечных целевых действий пользователя;
- стандартный трек взаимодействия с модулем может быть следующим (приведен пример для бота);

```
Добро пожаловать в функцонал модуля Пользователи.

**Для всех**
- Можно запросить роль "Ментора";

**Для админов**
- Можно утвердить или отменить запрос на роль "Ментора";

Примечание: само начало взаимодействия с ботом является фактором регистрации и авторизации. Для других ui (консоль, web-ui) могут быть логика регистрации и авторизации.
```

```
Добро пожаловать в функционал модуля Курсы.

Вес функционал делится по ролям.

**Для всех**
- Можно увидеть список менторов и их курсы;
- Можно ознакомиться с основной информацией курсов;
- Можно выбрать курс чтобы далее работать с ним (например в модуле "Поток курса")"
- Чтобы записать на курс используйте функционал модуля "Поток курса";

**Для Менторов**
- Можно добавить курс, определить его структуру и наполнить учебным материалом:
  - в начале создаем курс выбирая будет ли он основан на модули/проект/уроки или проект/уроки;
  - если выбрали модульный вариант:
    - добавляем модули;
    - добавляем дополнительную инфу к модулям;
  - добавляем проекты (к курсу или модулю в зависимости от типа курса);
  - добавляем дополнительную инфу к проектам;
  - добавляем уроки к проектам;
  - добавляем дополнительную инфу к урокам;
  - добавляем шаги к урокам;
- Чтобы запустить поток курса и управлять им используйте функционал модуля "Поток курса";
```

```
Добро пожаловать в функционал модуля Поток.

Здесь вы можете стать студентом курса или управлять курсом. Для начала работы с модулем вам нужно выбрать курс в модуле "Курс". Сейчас у вас выбран курс <course-label>.

**Для всех**
- Записаться на поток курса;
- Проходить курс;

**Для менторов**
- Можно запустить поток курса;
- Можно добавить студентов из списка записавшихся;
- Можно проводить поток;
```

- пользователь может после ознакомления с возможностями модуля увидеть список user-story и начать взаимодействие;
